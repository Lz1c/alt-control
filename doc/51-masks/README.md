# 51 · 面具条目

> 每个面具一个 md 文件。规则与模板见 [`../50-mask-rules.md`](../50-mask-rules.md)。
> 加新面具走 `/new-mask` 斜杠命令，会自动建文件 + 更新本索引。

## 命名

- 文件名 = 面具 ID，例如 `mask-dejiang-02.md`
- ID 规则：`mask-<体系 slug>-<两位数字>`，详见 [`../50-mask-rules.md § 面具编号命名`](../50-mask-rules.md)
