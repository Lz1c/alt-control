## 设定

- **民族 / 出处**：安顺地戏（汉军傩武将系）。详见 [`../50-mask-rules.md § 地域 / 体系参考清单`](../50-mask-rules.md)。
- **面具**：[`mask-dixi-01`](../51-masks/mask-dixi-01.md)（安顺地戏武将"奔面"，原创变体；盔面连体 + 雉尾 + 耳翅）
- **主色**：红 + 黑（武将相，参考五色体系"红=威慑/忠义"、"黑=刚直"）
- **外观**：
  - 肉眼**完全看不见**（项目核心机制）
  - 相机取景器里：身躯是穿着现代校园跑步服（旧款蓝白校训运动服 + 钉鞋）的高瘦男性，但**头部是一张活体化的安顺地戏武将面具**——盔甲样的额冠、瞋目、耳翅微微扇动；眼眶里没有眼球，是两团类似篝火的青蓝光点；嘴角因为长期奔跑微微上扬，露出一点不属于人的牙
  - 拍中达标后，整个鬼形态崩解，**头部凝固成实体木雕地戏面具**掉落在跑道上
- **叙事背景**：生前是该校田径队员；死因 TBD（实装阶段在 `60-levels/01-xiaoyuan-zhiye.md` 的剧情包装里定）

## 行为

Boss 战分**两阶段**，每阶段考查不同的快门特性。整体设计意图：L1 是"快门主题关"，单关讲透**冻结 / 长曝消形 / 摇镜 + 多形态显影**三种快门哲学。

### Phase 0 · 找鬼（长曝消形）

- 玩家走进操场，跑道上**飘着 5–8 个半透明的"夜训社团残影"**（持续做相同跑步动作的循环鬼影，**无害不可收服**，纯遮挡）
- Boss 此时**静止站在中圈**直视看台方向；但被残影遮挡，玩家肉眼看不见，1/250 普通拍照里也只见花花绿绿一片
- **解法**：玩家手动靠近→ 启用长曝（≥ 1s）→ **残影因持续移动被擦除**，露出 Boss 武将面孔正脸 → 视觉爆点 → 触发 Phase 1
- 这是**唯一进入 Phase 1 的方式**，不能跳

### Phase 1 · 拍鬼（多形态 + 摇镜）

- Boss 起步开跑，进入循环路径（操场跑道）+ 两档速度状态机
- **两档移动速度**（**无停止状态**）：
  - **巡跑档**：5 m/s（默认状态，绕跑道椭圆循环）
  - **冲刺档**：9 m/s（偶发，从弯道加速直线穿过主席台正前方约 3 秒）
- **多形态显影机制**：Boss 在不同快门档位下显示**不同程度的真相**——慢快门字面看到更多。详见下方"拍照判定要点"。

### 全程 AI 细节

- **冲刺哨音预告**：每次冲刺起步前 0.3 秒，脚步声戛然变重 + 一声运动场口哨"嗶——" → 给玩家时间举相机
- **冲刺路径固定**：冲刺总是从弯道直线穿过主席台正前方（玩家所在的看台正对面），固定路径让玩家可预判但难抓拍——也是摇镜法的最佳练习窗口
- **Phase 0 残影行为**：每只残影沿独立的 30s 循环路径跑，互相错位，整体形成"操场上一直有人在跑"的幻象

## 拍照判定要点

### 多形态显影（Phase 1 核心评分维度）

- Boss 在 **同一帧** 用 **不同快门** 拍下时显示**不同程度的真相**——慢快门字面看到更多。这是项目"相机能看穿"核心机制的可计量化
- 评分由 PhotoEvaluator 5 维基线 + **多形态显影加成** 共同决定（具体打分曲线待 `doc/31-photo-judgment.md` 实装）
- 三档形态：
  - **人形**（1/500+）：基础分，能拿 60 通关
  - **半显形**（1/125–1/500 + 任意拍法）：面具边缘可见，~70
  - **全显形**（≤ 1/60 + 摇镜跟随成功）：武将面 + 盔甲虚影 + 流线背景，~90 + 解锁隐藏成就 ★★★ **"追风"**

### 摇镜法判定（panning）

- **主体跟随评分**：相机角速度 vs Boss 角速度的匹配度（误差 ≤ 阈值 → 主体清晰）
- **背景流线评分**：背景模糊呈**放射状/水平线状**而非随机糊 → 加分
- 摇镜成功的视觉标志：取景器里 Boss 居中且清晰，操场跑道边线/看台拉成动感速度线

### 其他

- **三档任一达 60 → 通关**。糊弄打法仍可过，追风打法是技巧 + 显影双奖励
- **闪光灯**：L1 未解锁，本鬼无相关设计

## 失败反应

- **拍糊（不到 60 分）**：鬼**原地烟雾化消失 5 秒** → 重生在跑道随机一段（避免玩家蹲在原点重复拍）
- 鬼**不会追击玩家**（L1 软失败 + 温和型）
- 玩家**胶卷无限**，可无限 retry（L1 失败策略）

## 收服反馈

- 达标拍中那一刻：**时间慢放 1 秒** → 鬼形态从脚开始崩解为青蓝光雾 → 头部凝固成实体木雕地戏面具掉落到拍中位置 5m 内的跑道上
- 同时一声长哨"嗶——嘟"渐弱（呼应冲刺哨音预告）
- 玩家走过去拾取面具 → 关卡结算
- 面具拾取 → 回办公室自动挂上面具墙第 1 个空位

## 关联谜题

- 教学锁：**快门主题**（不只是"快门优先 = 拉快"）。L1 单关讲透三种快门哲学：
  - **冻结快速运动**（站定 + 1/500+，扰乱鬼"冲跑小孩"教）
  - **长曝消形 / 积累**（≥ 1s + 防抖，扰乱鬼"夜训残影"+"粉笔字鬼"教，Boss Phase 0 综合考）
  - **摇镜法 + 多形态显影**（1/60 + 跟随，扰乱鬼"白衣女"教，Boss Phase 1 综合考）
- 参考 [`../plan/knowledge-lock-chain.md § L1`](../plan/knowledge-lock-chain.md)
- L2 衔接钩子：冲刺档把 ISO 800 打满 → 玩家退场前体感"我的 ISO 不够用" → L2 ISO 教学动机
