# ghost-wu-li-can-ying · 雾里残影

🟡 In-Progress

**分类**：扰乱影
**所属段位**：段 2 · 雾带
**出现关卡**：[01 · 山](../60-levels/01-shan.md)
**教学概念**：长曝消形

## 设定

- **外观**：5-8 个半透明人影持续沿循环路径在雾里走过。雾后的中央有一个**静止的主体**（一个独自坐着的旅人 / 一头驮货的山骡 / 一具搁置的旧帐篷——TBD），被来回走动的残影遮挡
- **故事一句话**：早年迷路在雾里的登山队的影像，他们走啊走，永远找不到出路。中央的"主体"是这条队伍当初等待的人 / 物——一直没等到
- **朋友与它的关联**：朋友走过这里时雾很大，她自己研究出"长曝可以擦除残影"的拍法

## 行为

- **AI 模式**：5-8 个残影沿独立 30s 循环路径在雾里走，互相错位。中央主体完全静止
- **显形条件**：举相机进入雾带 + 在中央主体 8m 范围内
- **是否主动靠近玩家**：否（残影沿固定路径，主体静止）
- **是否伤害玩家**：否
- **拍照后反应**：≥ 1s 长曝拍下时，残影擦除 + 主体显形 + 笔记本完整版解锁；短快门拍下只见花花绿绿一片

## 机械数值

## 设计意图

- 教玩家：**慢快门 + 长曝消形**反直觉机制
- 长曝防抖（扶稳 buff）=（W2 / W3 实装项），玩家学会"找东西稳一下"
- 中央主体的"故事一句话"由玩家自己解读，不明说

## 诡异调味钩子

- **本影承担**：段 2 雾带是 SCP 风诡异层最密集的一段（3 处事件），不全发生在本影上，而是**段位附带**：
  - 长曝期间身后传来脚步声（被跟踪类）
  - 转身瞬间远处雾里有一个不动的人形轮廓，走近时消失（错位类）
  - 残影群之外偶尔有一个不属于该段的轮廓快速掠过画面边缘（错位类）
- **类型标签**（参考 [`../45-uncanny.md`](../45-uncanny.md)）：被跟踪 + 错位
- 详细事件描述（触发条件 / 视觉 / 音效 / 概率）见 [`../60-levels/01-shan.md § 段 2 诡异调味`](../60-levels/01-shan.md)
- 这些事件**不影响本影的拍照判定**——纯氛围层，玩家可以错过

## 技术要求（W3）

- 残影 alpha 衰减需要修改 `CAMPhotoCapture` 拍照管线，让长曝期间残影累积透明度
- 或用 `CAMExposureAccumulator` 真长曝积累 + 残影持续移动 → 自然擦除
