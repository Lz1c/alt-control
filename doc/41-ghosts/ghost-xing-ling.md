# ghost-xing-ling · 星灵

🟡 In-Progress

**分类**：扰乱影
**所属段位**：段 5 · 星脊（夜段）
**出现关卡**：[01 · 山](../60-levels/01-shan.md)
**教学概念**：长曝积累 / 星轨

## 设定

- **外观**：3-5 个漂浮的小光团，本身近乎透明（短曝下完全看不见）。在长曝光下逐渐显形为人脸 / 人形轮廓 / 鸟形等不固定形态。光色偏冷青蓝
- **故事一句话**：山顶夜里出现的影像，比山君更古早。可能是更早一批走过这条路的人寄存在星空里的痕迹
- **朋友与它的关联**：朋友在这里等过一整夜想拍星轨，没等到 30s 长曝就被风吹走了

## 行为

- **AI 模式**：缓慢漂浮（< 0.5 m/s）。**短快门拍下完全不显形**；长曝下随曝光时间累积亮度
- **显形条件**：举相机 + ≥ 1s 长曝（轮廓开始浮现）+ 4s 显形完整
- **是否主动靠近玩家**：否（自由漂浮在 5-15m 距离）
- **是否伤害玩家**：否
- **拍照后反应**：长曝拍达标 → 光团凝固一帧后向上飘升消失；短快门拍糊 → 无反应（玩家以为没东西）

## 机械数值

| 项 | 值 | 备注 |
|---|---|---|
| `blur_multiplier` | 0.2 | 慢漂浮，主要靠累积曝光显形 |

| 推荐光圈 | f/4 | 段 5 固定 |
| 推荐 ISO | 400 – 800 | 长曝下夜空亮度足够 |
| 最低对焦距离 | 5 m | |
| 亮度偏好 | 极暗 | 无月夜 |
| 占屏比要求 | ≥ 8% | 多个小光团 |
| 防抖要求 | 三脚架 / 扶稳必需 | 长曝抖动直接糊 |
| **passingScore** | **60** | 显形 + 不糊即可 |

## 设计意图

- 教玩家：**长曝积累机制**——短快门看不见的东西，长曝下显形
- 让玩家**等**——一次拍照真的要 8 秒，是节奏的转折，让玩家慢下来
- 与段 6 顶峰守者构成节奏对比：星脊静夜 → 顶峰云海

## 技术要求（W3 必装）

- **`CAMExposureAccumulator` 接入 `CAMPhotoCapture` 主拍照管线**：当前代码累积器已有但未接入。星灵显形机制是这一关唯一依赖真长曝积累的设计
- 长曝期间相机抖动惩罚需放大（玩家应主动找三脚架点 / 平台扶稳）
