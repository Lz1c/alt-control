# ghost-qintong · 秦童

**分类**：扰乱鬼（世俗丑角型 · **刻意打破恐怖基调**）
**出现关卡**：几乎所有关卡（高频率、低威胁、背景噪点式分布）

## 设定
- 民族/出处：德江傩堂戏"秦童"——傩戏里的歪嘴丑角，与地盘和尚、梁九同属**傩的世俗幽默面**
- 面具：`mask-dejiang-05`（歪嘴、暴牙、扯眼皮；红黑面底配金色小饰；保留原型秦童的诙谐感，**不做狰狞化处理**）
- 外观：穿着褪色戏班衣，腰间挂一串铜铃；走路一踢踏一踢踏，经常蹲在关卡角落打哈欠、抓虱子，或者原地模仿玩家刚才的姿势
- 叙事背景：戏班散了之后留下来的傩戏杂角。他知道自己是假的，也知道玩家在演戏——他看玩家看得津津有味

## 行为
- AI 模式：观察玩家 + 低频率模仿（玩家举相机 → 他隔 2 秒也举一个看不见的相机；玩家蹲下 → 他也蹲下）
- 干扰方式：几乎不干扰；**当相机瞄准他超过 3 s，他会摘下面具——露出底下一张一模一样的秦童面具**——可循环无限次，可连拍捕捉
- 移动速度：0.5 m/s
- 会否攻击玩家：否；**遭遇他应当让玩家短暂松一口气**

## 机械数值
| 项 | 值 |
|---|---|
| `blur_multiplier` | 1.0 |
| 图鉴解锁条件 | 在他"摘面具"的循环中连拍 ≥ 3 张，每张面具清晰（`passingScore` ≥ 40） |
| 给予奖励 | 图鉴条目 × 1、"秦童铜铃"（办公室书架摆件，轻摇有声） |
| 亮度偏好 | 中 / 亮 |

## 设计意图
- **节奏调味**：专门对冲"全员恐怖"的单调（参见 [`../50-mask-rules.md`](../50-mask-rules.md) 三大风险之二：避免东方主义式单色恐怖）
- 文化提醒：傩不等于狞厉，傩班里有丑角、有世俗、有生活情趣
- 功能教学：连拍节奏、"相机瞄准触发 NPC 反应"——为后面更复杂的反应式 Boss 做铺垫
- 彩蛋位：秦童摘到第 7 次的时候偶尔会露出**真的脸**（0.3 s 窗口），拍到可解锁隐藏图鉴
