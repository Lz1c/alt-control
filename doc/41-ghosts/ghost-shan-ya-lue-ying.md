# ghost-shan-ya-lue-ying · 山鸦掠影

🟡 In-Progress

**分类**：扰乱影
**所属段位**：段 3 · 岩壁
**出现关卡**：[01 · 山](../60-levels/01-shan.md)
**教学概念**：快门冻结 + ISO 基础

## 设定

- **外观**：3-5 只半透明乌鸦循环掠过岩壁前，速度极快（~ 8-10 m/s 横向）。羽毛虚化拖出短尾迹
- **故事一句话**：山鸦本身。山记得每一只飞过的鸟，重复着它们曾经的飞行路径
- **朋友与它的关联**：朋友研究过怎么拍清这种快速横向运动 — "1/1000 才冻得住"

## 行为

- **AI 模式**：沿弧形路径循环掠过岩壁前（8-10s 一轮）
- **显形条件**：举相机进入观景台范围 + 半按命中飞行路径中的乌鸦
- **是否主动靠近玩家**：否（弧形路径距玩家 8-15m）
- **是否伤害玩家**：否
- **拍照后反应**：达标 → 乌鸦在画面里凝固一帧 + 一道光飘起 + 影继续飞但不再循环；糊 → 该轮飞过后 5 秒回来重飞

## 机械数值

| 项 | 值 | 备注 |
|---|---|---|
| `blur_multiplier` | **2.5** | 高速横向，不冻结就糊 |

| 推荐快门 | **1/1000 s** | 慢一档明显糊 |

| **passingScore** | **60** | |

## 设计意图

- 教玩家：**快快门冻结快速运动** + **ISO 是亮度补偿手段**
- ISO 800 解锁伏笔 Boss A 段 4 的 ISO 噪点考察
- 失败成本低（鸟会重飞），鼓励玩家尝试不同档位
