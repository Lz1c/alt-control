# ghost-xiaomian · 笑面娃娃

**分类**：扰乱鬼
**出现关卡**：02、05、08

## 设定
- 民族/出处：毛南族肥套"花林仙官（小婆王）"的扭曲化
- 面具：`mask-maonan-03`（白底、浅粉红腮、眉心朱砂；原型是慈祥可爱，游戏里把嘴角咧到耳根，花冠换成干枯的纸扎小人）
- 外观：身高仅 1.1 m，赤脚，手里拎着一个与她等身的纸人；纸人的脸和她一模一样
- 叙事背景：被烧掉的"替身纸人"。一户求子家族在还愿仪式上把她献出之后，她没按规矩走掉

## 行为
- AI 模式：藏在关卡场景物件背后（柜子、水缸、神龛、供桌、旧行李箱）
- 干扰方式：持续发出童谣 + 间歇笑声（定向 3D 音源，位置随她藏身物体），画面里不直接出现；**玩家把相机对焦到她所藏的物件**时，她才从物件侧后方慢慢浮出 2 秒，再消失
- 移动速度：0 m/s（触发后静止显形，不追击）
- 会否攻击玩家：否；但她的童谣会逐渐与关卡 BGM 同步、错拍半拍，制造听觉不适

## 机械数值
| 项 | 值 |
|---|---|
| `blur_multiplier` | 0.6（显形期静止，容错宽松） |
| 触发条件 | `CAMFocusController` 对焦距离 = 她藏身物件距离 ± 0.3 m，且连续保持 ≥ 1.5 s |
| 图鉴解锁条件 | 在她浮出的 2 秒窗口拍到正面 + 占屏比 ≥ 8% |
| 给予奖励 | 图鉴条目 × 1 |
| 亮度偏好 | 暗 |

## 设计意图
- 把"对焦"这个相对被动的操作变成**主动触发器**——玩家必须真的理解对焦在干什么
- 用声音做空间定位教学：靠耳朵判断物件方位，再靠对焦验证
- 美学反差：把毛南族"慈祥"的面具推到过度，反而更毛骨悚然（呼应 [`../50-mask-rules.md`](../50-mask-rules.md) 的"雅俗并置"与"避免东方主义 = 单色恐怖"准则，但承认过度的慈祥本身就是一种恐怖）
