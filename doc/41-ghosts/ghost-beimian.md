# ghost-beimian · 背面

**分类**：扰乱鬼
**出现关卡**：03、05

## 设定
- 民族/出处：苗族巴岱（白色法巾意象）+ 汉俗白无常混合
- 面具：`mask-han-02`（白面无眼珠、嘴角微微下抿；面具下方缝一小条金箔，暗示曾有神格，如今只剩亡灵）
- 外观：白长衫拖地，面具永远朝向玩家；**它始终站在玩家身后 2–3 m**，且不会被直视到
- 叙事背景：生前被人从背后偷袭而死的人；死后不愿再把背部留给世界，于是永远站在别人的身后

## 行为
- AI 模式：锁定玩家朝向的反向 180°；玩家转头时瞬移到新的身后
- 干扰方式：脚步声、衣料摩擦声始终从身后传来；**肉眼直视它的一瞬间它必然已经不在**
- 移动速度：瞬移（等价 ∞ m/s，但每次瞬移之间有 0.15 s 位移期可被高速快门逮到一点影子）
- 会否攻击玩家：否；但它的存在会让关卡的恐惧音（心跳、低频嗡鸣）持续抬升

## 机械数值
| 项 | 值 |
|---|---|
| `blur_multiplier` | 4.0（瞬移期速度向量极高，直接拍正面必糊成一道白条） |
| 显形条件 | **只能通过反射面间接拍到**：窗玻璃、水面、金属擦亮面、镜面、玻璃橱柜 |
| 图鉴解锁条件 | 一张照片中，反射面上识别出面具区域 + 对焦误差 ≤ 40% + 反射面占屏比 ≥ 5% |
| 给予奖励 | 图鉴条目 × 1、"镜之线索"碎片 × 1（办公室档案页） |
| 亮度偏好 | 暗 / 极暗 |

## 设计意图
- 把"镜面反射"正式作为一种拍摄手法引入教学；引导玩家主动巡场景找反射面
- 与 [`ghost-yingzi`](ghost-yingzi.md)（余光影子）区分：影子是一闪而过、瞬发刺激；背面是**恒定存在、拒绝被正视**的长期压迫
- 玩家不能消灭它——但拍到它一次，它会"安静"5 分钟（音效衰减），做成可交易的心理空间
