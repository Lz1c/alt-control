# ghost-feng-kou-shan-jun · 风口山君

🟡 In-Progress

**分类**：Boss 影
**所属段位**：段 4 · 风口 / 雪线
**首次出现关卡**：[01 · 山](../60-levels/01-shan.md)
**教学概念**：综合（快门 × ISO 噪点）

## 设定

- **外观**：巨大人影（身高接近 3m），背对玩家立在风口正中。穿着旧式厚重登山装，背包绑带在风里飘。脸朝向风暴中心，看不见。轮廓在风暴粒子里时隐时现
- **故事一句话**：一个独自上过这座山的登山者，最后没下来。山记住了他立在风口的样子——很多年，一直站着
- **朋友与它的关联**：朋友也在这里试过拍它，承认自己拍糊了。这是朋友笔记本里少有的"失败"页

## 行为

- **AI 模式**：完全静止，背对玩家立在风口正中
- **显形条件**：举相机进入风口背风处 + 半按对焦命中
- **环境干扰**：风暴粒子 + 暗光 + 玩家相机轻微抖动（风的物理影响）
- **是否主动靠近玩家**：否
- **是否伤害玩家**：否
- **拍照后反应**：达标 → 影缓缓侧脸看向玩家方向（不正脸），一道光飘起，淡出；糊 → 风暴粒子加重 5 秒后回到原状

## 机械数值

| 项 | 值 | 备注 |
|---|---|---|
| `blur_multiplier` | 1.0 | 影本身静止，但风暴粒子 + 抖动让画面整体糊 |
| 推荐快门 | **1/500 s** | 冻结风暴粒子的下限 |
| 推荐光圈 | f/4 | 段 4 固定 |
| 推荐 ISO | **1600 – 3200** | 暗光 + 1/500 + f/4 必需高 ISO；伴随明显噪点 |
| 最低对焦距离 | 8 m | 影立在风口中心，玩家在背风处 |
| 亮度偏好 | 极暗 | EV 4-5 |
| 占屏比要求 | ≥ 12% | |
| **passingScore** | **65** | 综合考门槛 |

## 拍照判定要点

- **必须冻结风暴粒子**：1/500+ 才不糊
- **ISO 噪点权衡**：ISO 1600 噪点开始明显，3200 噪点饱和。`PhotoEvaluator` 噪点维度扣分但非否决
- **占屏比** ≥ 12% 即可，不强求贴近

## 拍达标奖励

- 笔记本第 4 页**完整版** + **隐藏页解锁**：
  > "我也没拍成。谢谢。"
  > [朋友的一张山顶手记照：清晨的薄雾]

## 失败反应

- 拍糊 → 风暴粒子加重 5 秒，影在视觉上更"糊" → 5 秒后回到原状，玩家可继续尝试
- **Boss A 全程不拍也不阻挡推进** → 玩家可直接走过去进入段 5

## 设计意图

- **整段山路上第一次让玩家在恶劣条件下做参数权衡**：快门冻结 vs ISO 噪点
- 朋友"承认失败" + 隐藏奖励"我也没拍成，谢谢" = v2 治愈基调的高潮之一
- Boss A 不阻挡推进 = 玩家如果技术不够好仍能登顶（中性结局），不在技能上设关
