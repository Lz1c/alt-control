# ghost-lin-ci-jing-zuo · 林祠静坐者

🟡 In-Progress

**分类**：扰乱影
**所属段位**：段 1 · 林线
**出现关卡**：[01 · 山](../60-levels/01-shan.md)
**教学概念**：半按对焦 + 中央构图（破冰）

## 设定

- **外观**：人形，背对玩家盘坐在小山祠前。穿着不属于这个时代的旧式登山装（粗布外套 / 旧斗笠 / 帆布背包）。半透明，发丝随风但身体不动。
- **故事一句话**：曾在这里停下来祈愿过的某个登山者。年代不明。
- **朋友与它的关联**：朋友也在这里拍过它，但构图全没把握好——把人挤到画面边缘。

## 行为

- **AI 模式**：完全静止，背对玩家盘坐
- **显形条件**：玩家举相机进入小山祠 5m 范围内 + 半按对焦命中
- **是否主动靠近玩家**：否
- **是否伤害玩家**：否
- **拍照后反应**：拍达标 → 一道光从影身上飘起，影向玩家方向轻轻点头后淡出；拍糊 → 影 3 秒淡出后重新显形

## 设计意图

- 教玩家：**举相机 → 半按对焦 → 中央构图 → 全按拍照**的最基础流程
- 静止 + 宽容数值 = 玩家几乎一定能拍中达标，建立成功感
- 朋友的字 = 第一次让玩家"听到"朋友的声音，建立情感锚点
