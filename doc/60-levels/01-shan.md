# 01 · 山（v2 唯一关卡）

🟡 In-Progress

## 基本信息

- **预计时长**：30-45 分钟
- **结局**：完整 / 中性
- **场景**：Cairn 借用资产（具体山系部分 TBD，见 `plan/open-questions.md`）
- **通关条件**：登顶 + 至少举相机看了一次 Boss B
- **完整结局判定**：Boss B `passingScore` ≥ 70

## 整体结构

```
主菜单 → trailhead 开场 → 段 1-6 → 顶峰 → 相册回放 → 黑屏献给字幕 → 重玩按钮
```

线性、单向、不可回头下山（地形挡板 + 软引导）。每段进入时**朋友的笔记本自动浮现一页**，教这一段需要的相机概念 + 解锁该参数旋钮。

## trailhead 开场

- **位置**：山脚营地，Cairn 第一段山路起点
- **触发**：游戏开始进入
- **演出**：
  1. 黑屏 → 主角坐在背包旁，背包里掉出朋友的相机 + 笔记本
  2. UI 提示：捡起相机（E）→ 装备相机
  3. UI 提示：举起相机（Q）→ 进入取景器视角，看到山的轮廓
  4. UI 提示：半按对焦（O）+ 全按拍照（P）→ 拍下第 1 张测试照（**进相册存盘**）
  5. 主角内心独白 1 句（文案 TBD：例 "他说山顶能看到云海。我答应过他要拍下来。"）
- **解锁旋钮**：举起相机 / 半按对焦 / 全按拍照
- **解锁参数**：快门 1/250 – 1/60 / ISO 100-400 / 光圈固定 f/4

## 段 1 · 林线

- **影**：[`ghost-lin-ci-jing-zuo`](../41-ghosts/ghost-lin-ci-jing-zuo.md)（林祠静坐者）
- **地形**：缓坡林间，光斑斑驳，途经一座小山祠（Cairn 资产里的小型石建筑）
- **建议拍点**：山祠正面 5m
- **笔记本第 1 页**：构图取景框 + 半按对焦
  - 朋友的字迹："最简单的是把人放中间。我每次都犯这毛病——你别学我。"
  - 朋友样片：林祠半身构图（缺角版 / 完整版）
- **影行为**：静坐祠前，背对玩家，举相机时显形。拍达标 → 一道光从影身上飘起，影向玩家方向轻轻点头后淡出
- **passingScore**：50

## 段 2 · 雾带

- **影**：[`ghost-wu-li-can-ying`](../41-ghosts/ghost-wu-li-can-ying.md)（雾里残影）
- **地形**：缓坡 → 雾带，能见度 15-20m。需要慢快门"看穿"雾后的真相
- **建议拍点**：雾带中部一块石头旁
- **笔记本第 2 页**：慢快门刻度 + 长曝消形原理
  - 朋友的字迹："雾里有东西。1/30 拍是糊的。1.2s 才显形。注意手不能抖——我用石头当三脚架。"
  - 朋友样片：雾里残影长曝缺角版 / 完整版
- **影行为**：5-8 个半透明循环移动的"残影"持续走过，挡住中央的"主体"。≥1s 长曝时残影擦除，主体显形（一个静坐的人或物）
- **解锁参数**：快门下限放到 **1s**
- **passingScore**：60

## 段 3 · 岩壁

- **影**：[`ghost-shan-ya-lue-ying`](../41-ghosts/ghost-shan-ya-lue-ying.md)（山鸦掠影）
- **地形**：开阔岩壁段，强光逆光。山鸦群从岩壁上方横向掠过
- **建议拍点**：岩壁前观景台
- **笔记本第 3 页**：快快门 + ISO 基础
  - 朋友的字迹："鸟飞得很快。我用 1/1000 + ISO 200 才冻得住。慢一档全是糊的。"
  - 朋友样片：山鸦冻结版 / 糊版
- **影行为**：山鸦循环掠过岩壁前。`blur_multiplier` = 2.5（高速横向）。1/500+ 才能拍清
- **解锁参数**：快门上限放到 **1/2000**；ISO 上限放到 **800**
- **passingScore**：60

## 段 4 · 风口（Boss A）

- **影**：[`ghost-feng-kou-shan-jun`](../41-ghosts/ghost-feng-kou-shan-jun.md)（风口山君）
- **地形**：风口 / 雪线，暴风粒子环境，暗光（EV 4-5）
- **建议拍点**：风口背风处的岩石遮蔽
- **笔记本第 4 页**：ISO 800-3200 + 噪点权衡
  - 朋友的字迹："风太大了。我用 1/500 + ISO 1600 拍。噪点很重，但拍清了。我没拍成完整的它。如果你能拍下来——就替我看看它的脸。"
  - 朋友样片：风口山君糊版（朋友承认失败的版本）
- **影行为**：巨大人影背对玩家，立在风口正中。风暴里轮廓时隐时现。需要 1/500+ 快门冻结风暴粒子 + ISO 1600+ 提亮
- **解锁参数**：ISO 上限放到 **3200**
- **passingScore**：65
- **拍达标奖励**：笔记本隐藏页 "我也没拍成。谢谢。"
- **拍糊**：影短暂淡出 5 秒重新显形；Boss A 即使全程拍糊**不阻挡推进**

## 段 5 · 星脊

- **影**：[`ghost-xing-ling`](../41-ghosts/ghost-xing-ling.md)（星灵）
- **地形**：山脊夜段，星空铺满半边天，无月光
- **建议拍点**：山脊上一处天然平台（可放三脚架）
- **笔记本第 5 页**：长曝积累 / 星轨拍法
  - 朋友的字迹："这里我等了一整夜。8s 曝光，星星会拖出小尾巴。30s 就是星轨。我没等到 30s——风太大了。"
  - 朋友样片：星灵 + 星轨（短曝缺角 / 长曝完整）
- **影行为**：星灵漂浮在山脊上方，本身近乎透明，**只有长曝光下才显形**（≥ 1s 开始有形，≥ 4s 显形完整）
- **解锁参数**：快门下限放到 **8s**
- **passingScore**：60
- **技术要求**：`CAMExposureAccumulator` 接入 `CAMPhotoCapture` 拍照管线（W3 实装项）

## 段 6 · 顶峰（Boss B）

- **影**：[`ghost-shan-ding-shou-zhe`](../41-ghosts/ghost-shan-ding-shou-zhe.md)（山顶守者）
- **地形**：山顶平台，云海铺展在脚下，极远景
- **建议拍点**：山顶石碑前
- **笔记本第 6 页**：手动对焦 + 大光圈景深
  - 朋友的字迹（最后一段）："这是我想拍的画面。守者站在云海前面，光从他背后来。我想用 f/2.8 + 手动对焦在他的肩膀上，让云海虚化掉。我没拍成。剩下的交给你了。"
  - 朋友样片：山顶守者构图缺角版（焦点不准 / 光圈不对）
- **影行为**：守者背对玩家，立在山顶石碑旁，凝视云海方向。**不动**、不主动转身。**朋友想看到的画面就是从守者背后看出去的云海光**
- **解锁参数**：手动对焦 + 全光圈范围 **f/1.4 – f/16**
- **passingScore**：70
- **拍达标 → 完整结局**：守者缓缓转头，正脸朝向玩家方向（玩家不需要再拍这一帧，叙事自然展开）→ 主角合上相机 → 进入相册回放
- **拍糊 → 中性结局**：守者保持背对，云海一片白光罩住画面 → 主角合上相机 → 进入相册回放

## 相册回放（结尾）

- **触发**：玩家在山顶按下 Boss B 的拍照（无论达标）
- **演出**：
  1. 黑屏 → 相册自动翻页，按时间序播放本次拍的所有 PNG
  2. 每张照片停留 1.5 秒，平移过渡
  3. 完整结局：相册结尾插入 Boss B 守者转头那张达标照片，停留 4 秒
  4. 中性结局：相册结尾停在一张白光照
  5. 黑屏 → 字幕 "献给 ___ "（朋友名 TBD）
  6. 完整结局：补一段朋友最后一封信文案（TBD）
  7. 字幕滚屏 → 主菜单（"重玩 / 退出"）

## 失败策略（v2 治愈基调）

| 玩家行为 | 反应 |
|---|---|
| 拍糊 | 影短暂淡出 3-5 秒后重新显形；笔记本只解锁半页 |
| 跳过某段的影 | 笔记本半页提示"这里你略过了"，可在该段范围内回头补拍 |
| Boss A 全程不拍 | 不阻挡推进，正常进入段 5 |
| Boss B 拍糊 | 进相册回放（中性结局）|
| 关游戏 | 重新开始时从 trailhead 重新走（v2 不做存档续玩）|

**没有**：胶卷消耗 / Game Over 屏 / 重试 prompt / 影追击 / 死亡 / 跌落 / jump scare。
