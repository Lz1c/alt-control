# 01 · 山（v2 唯一关卡）

🟡 In-Progress

## 基本信息

- **预计时长**：30-45 分钟
- **结局**：完整 / 中性
- **场景**：Cairn 借用资产（具体山系部分 TBD，见 `plan/open-questions.md`）
- **通关条件**：登顶 + 至少举相机看了一次 Boss B
- **完整结局判定**：Boss B `passingScore` ≥ 70

## 整体结构

```
主菜单 → trailhead 开场 → 段 1-6 → 顶峰 → 相册回放 → 黑屏献给字幕 → 重玩按钮
```

线性、单向、不可回头下山（地形挡板 + 软引导）。每段进入时**朋友的笔记本自动浮现一页**，教这一段需要的相机概念 + 解锁该参数旋钮。

## trailhead 开场

- **位置**：山脚营地，Cairn 第一段山路起点
- **触发**：游戏开始进入
- **演出**：
  1. 黑屏 → 主角坐在背包旁，背包里掉出朋友的相机 + 笔记本
  2. UI 提示：捡起相机（E）→ 装备相机
  3. UI 提示：举起相机（Q）→ 进入取景器视角，看到山的轮廓
  4. UI 提示：半按对焦（O）+ 全按拍照（P）→ 拍下第 1 张测试照（**进相册存盘**）
  5. 主角内心独白 1 句（文案 TBD：例 "他说山顶能看到云海。我答应过他要拍下来。"）
- **解锁旋钮**：举起相机 / 半按对焦 / 全按拍照
- **解锁参数**：快门 1/250 – 1/60 / ISO 100-400 / 光圈固定 f/4

## 段 1 · 林线

- **影**：[`ghost-lin-ci-jing-zuo`](../41-ghosts/ghost-lin-ci-jing-zuo.md)（林祠静坐者）
- **地形**：缓坡林间，光斑斑驳，途经一座小山祠（Cairn 资产里的小型石建筑）
- **建议拍点**：山祠正面 5m
- **笔记本第 1 页**：构图取景框 + 半按对焦
  - 朋友的字迹："最简单的是把人放中间。我每次都犯这毛病——你别学我。"
  - 朋友样片：林祠半身构图（缺角版 / 完整版）
- **影行为**：静坐祠前，背对玩家，举相机时显形。拍达标 → 一道光从影身上飘起，影向玩家方向轻轻点头后淡出
- **passingScore**：50

## 段 2 · 雾带

- **影**：[`ghost-wu-li-can-ying`](../41-ghosts/ghost-wu-li-can-ying.md)（雾里残影）
- **地形**：缓坡 → 雾带，能见度 15-20m。需要慢快门"看穿"雾后的真相
- **建议拍点**：雾带中部一块石头旁
- **笔记本第 2 页**：慢快门刻度 + 长曝消形原理
  - 朋友的字迹："雾里有东西。1/30 拍是糊的。1.2s 才显形。注意手不能抖——我用石头当三脚架。"
  - 朋友样片：雾里残影长曝缺角版 / 完整版
- **影行为**：5-8 个半透明循环移动的"残影"持续走过，挡住中央的"主体"。≥1s 长曝时残影擦除，主体显形（一个静坐的人或物）
- **解锁参数**：快门下限放到 **1s**
- **passingScore**：60

## 段 3 · 岩壁

- **影**：[`ghost-shan-ya-lue-ying`](../41-ghosts/ghost-shan-ya-lue-ying.md)（山鸦掠影）
- **地形**：开阔岩壁段，强光逆光。山鸦群从岩壁上方横向掠过
- **建议拍点**：岩壁前观景台
- **笔记本第 3 页**：快快门 + ISO 基础
  - 朋友的字迹："鸟飞得很快。我用 1/1000 + ISO 200 才冻得住。慢一档全是糊的。"
  - 朋友样片：山鸦冻结版 / 糊版
- **影行为**：山鸦循环掠过岩壁前。`blur_multiplier` = 2.5（高速横向）。1/500+ 才能拍清
- **解锁参数**：快门上限放到 **1/2000**；ISO 上限放到 **800**
- **passingScore**：60

## 段 4 · 风口（Boss A）

- **影**：[`ghost-feng-kou-shan-jun`](../41-ghosts/ghost-feng-kou-shan-jun.md)（风口山君）
- **地形**：风口 / 雪线，暴风粒子环境，暗光（EV 4-5）
- **建议拍点**：风口背风处的岩石遮蔽
- **笔记本第 4 页**：ISO 800-3200 + 噪点权衡
  - 朋友的字迹："风太大了。我用 1/500 + ISO 1600 拍。噪点很重，但拍清了。我没拍成完整的它。如果你能拍下来——就替我看看它的脸。"
  - 朋友样片：风口山君糊版（朋友承认失败的版本）
- **影行为**：巨大人影背对玩家，立在风口正中。风暴里轮廓时隐时现。需要 1/500+ 快门冻结风暴粒子 + ISO 1600+ 提亮
- **解锁参数**：ISO 上限放到 **3200**
- **passingScore**：65
- **拍达标奖励**：笔记本隐藏页 "我也没拍成。谢谢。"
- **拍糊**：影短暂淡出 5 秒重新显形；Boss A 即使全程拍糊**不阻挡推进**

## 段 5 · 星脊

- **影**：[`ghost-xing-ling`](../41-ghosts/ghost-xing-ling.md)（星灵）
- **地形**：山脊夜段，星空铺满半边天，无月光
- **建议拍点**：山脊上一处天然平台（可放三脚架）
- **笔记本第 5 页**：长曝积累 / 星轨拍法
  - 朋友的字迹："这里我等了一整夜。8s 曝光，星星会拖出小尾巴。30s 就是星轨。我没等到 30s——风太大了。"
  - 朋友样片：星灵 + 星轨（短曝缺角 / 长曝完整）
- **影行为**：星灵漂浮在山脊上方，本身近乎透明，**只有长曝光下才显形**（≥ 1s 开始有形，≥ 4s 显形完整）
- **解锁参数**：快门下限放到 **8s**
- **passingScore**：60
- **技术要求**：`CAMExposureAccumulator` 接入 `CAMPhotoCapture` 拍照管线（W3 实装项）

## 段 6 · 顶峰（Boss B）

- **影**：[`ghost-shan-ding-shou-zhe`](../41-ghosts/ghost-shan-ding-shou-zhe.md)（山顶守者）
- **地形**：山顶平台，云海铺展在脚下，极远景
- **建议拍点**：山顶石碑前
- **笔记本第 6 页**：手动对焦 + 大光圈景深
  - 朋友的字迹（最后一段）："这是我想拍的画面。守者站在云海前面，光从他背后来。我想用 f/2.8 + 手动对焦在他的肩膀上，让云海虚化掉。我没拍成。剩下的交给你了。"
  - 朋友样片：山顶守者构图缺角版（焦点不准 / 光圈不对）
- **影行为**：守者背对玩家，立在山顶石碑旁，凝视云海方向。**不动**、不主动转身。**朋友想看到的画面就是从守者背后看出去的云海光**
- **解锁参数**：手动对焦 + 全光圈范围 **f/1.4 – f/16**
- **passingScore**：70
- **拍达标 → 完整结局**：守者缓缓转头，正脸朝向玩家方向（玩家不需要再拍这一帧，叙事自然展开）→ 主角合上相机 → 进入相册回放
- **拍糊 → 中性结局**：守者保持背对，云海一片白光罩住画面 → 主角合上相机 → 进入相册回放

## 相册回放（结尾）

- **触发**：玩家在山顶按下 Boss B 的拍照（无论达标）
- **演出**：
  1. 黑屏 → 相册自动翻页，按时间序播放本次拍的所有 PNG
  2. 每张照片停留 1.5 秒，平移过渡
  3. 完整结局：相册结尾插入 Boss B 守者转头那张达标照片，停留 4 秒
  4. 中性结局：相册结尾停在一张白光照
  5. 黑屏 → 字幕 "献给 ___ "（朋友名 TBD）
  6. 完整结局：补一段朋友最后一封信文案（TBD）
  7. 字幕滚屏 → 主菜单（"重玩 / 退出"）

## 失败策略（v2 治愈基调）

| 玩家行为 | 反应 |
|---|---|
| 拍糊 | 影短暂淡出 3-5 秒后重新显形；笔记本只解锁半页 |
| 跳过某段的影 | 笔记本半页提示"这里你略过了"，可在该段范围内回头补拍 |
| Boss A 全程不拍 | 不阻挡推进，正常进入段 5 |
| Boss B 拍糊 | 进相册回放（中性结局）|
| 关游戏 | 重新开始时从 trailhead 重新走（v2 不做存档续玩）|

**没有**：胶卷消耗 / Game Over 屏 / 重试 prompt / 影追击 / 死亡 / 跌落。

> ⚠️ 关于 jumpscare：v2 调味剂决议放宽了反设计——全片**保留 1 处真扑面**（段 4 风口 Boss A 首次命中）。详见 [`../45-uncanny.md`](../45-uncanny.md)。

## 诡异调味事件清单（SCP 风）

> v2 治愈基调下的**少量恐怖元素**。规则全文见 [`../45-uncanny.md`](../45-uncanny.md)。
> **硬约束**：山顶 Boss B 段绝对干净；全片只 1 处真扑面；概率触发，治愈玩家可完全错过。

### trailhead · 计数错乱

- **事件**：玩家拍下第一张测试照后，相册里偶尔多 1 张玩家不记得拍过的照片（一张山的轮廓 / 一张陌生人背影）
- **触发**：5% 概率，每周目最多 1 次
- **类型**：照片诡异 / 计数错乱
- **音效**：无（沉默是关键）

### 段 1 林线 · 观察反转

- **事件**：玩家拍完林祠静坐者后离开几步，回头**用肉眼**瞄一眼山祠——静坐者的轮廓还在那里（半透明）。**举起相机看**——位置空了。再放下相机用肉眼看——又有了。来回 1-2 次后停止
- **触发**：拍完林祠静坐者后离开 ≥ 5m + 玩家自然回头，10% 概率
- **类型**：观察反转（**反转 v2 核心机制**）
- **音效**：无；只有玩家自己的呼吸轻一拍
- **设计意图**：让玩家 doubt 自己。"是我看错了吗？是我在举相机时反而看不见？"

### 段 2 雾带 · 三层叠加

#### a. 被跟踪 · 长曝期间脚步声

- **事件**：玩家长曝拍雾里残影时，身后 3-5m 传来脚步声（缓慢、不规律），从右后到左后；玩家转身就停
- **触发**：长曝 ≥ 1s 期间，30% 概率（每段路径上最多 1 次）
- **类型**：被跟踪
- **音效**：3D 立体声脚步在雪 / 落叶上，渐近渐远

#### b. 错位 · 远处静立人形

- **事件**：玩家自然转身（任意原因）瞬间，雾里 15-20m 处有一个不动的人形轮廓站着；玩家迈步靠近 2 步内消失
- **触发**：玩家转身角速度 > 阈值，20% 概率
- **类型**：错位
- **音效**：无

#### c. 错位 · 边缘掠过

- **事件**：玩家拍照那一帧，画面边缘**取景器外侧**有一个轮廓快速掠过（不是雾里残影群里的成员）
- **触发**：每次拍照，5% 概率
- **类型**：错位
- **音效**：极低音量布料摩擦声

### 段 3 岩壁 · 反向掠过

- **事件**：山鸦群循环掠过 3-4 轮后，偶尔出现 1 只**反向飞**的鸦（从原本飞出的方向飞回来），动作流畅但路径不在循环模型里
- **触发**：玩家在该段停留 ≥ 30s，每 30s 检查一次 15% 概率
- **类型**：错位
- **音效**：鸦鸣，但音调比正常的略低半度

### 段 4 风口 · 真扑面 + 重影

#### a. 真扑面 jumpscare（**全片唯一**）

- **事件**：玩家**首次**半按对焦命中风口山君时，山君瞬移到玩家面前 1.5m，正脸闪现 1-2 帧（眼神空洞 / 嘴张大 / 占屏 ~50%），低吼一声，风暴粒子吃掉，回到原位背对
- **触发**：固定一次性，进段 4 后第一次半按对焦命中山君
- **类型**：jumpscare
- **音效**：低频吸气 + 一记低吼（< 0.4s 短促刺响；唯一允许的刺响）
- 详见 [`../41-ghosts/ghost-feng-kou-shan-jun.md § 扑面 jumpscare`](../41-ghosts/ghost-feng-kou-shan-jun.md)

#### b. 错位 · 雪线第二剪影

- **事件**：玩家从风口背风处望向远处雪线（≥ 30m），偶尔看到**第二个山君的剪影**——同样背对、同样姿态、低细节远景。玩家定睛 1-2 秒后消失
- **触发**：玩家朝雪线方向看 ≥ 2s，10% 概率
- **类型**：错位（重影）
- **音效**：风声里混入断续的"哼"（不是音乐，是人哼）

### 段 5 星脊 · 双层

#### a. 照片诡异 · 边缘多一个星灵

- **事件**：玩家拍下星轨 PNG 后，画面边缘偶尔多出 1 个**没显形**的星灵剪影（实际拍照过程中并没有那个星灵显形）
- **触发**：拍下任一张含星灵的长曝照，20% 概率
- **类型**：照片诡异
- **音效**：无（PNG 已存盘后玩家自己发现）

#### b. 声音错位 · 远方林祠低语

- **事件**：玩家在星脊抬头看星空 ≥ 5s 时，远方传来段 1 林祠那种诵经低语（极低音量，听不清内容）
- **触发**：玩家抬头视角 + 静止 ≥ 5s，15% 概率
- **类型**：声音错位
- **音效**：低音量、低频、不清晰的人声哼唱

### 段 6 顶峰 · **绝对干净**

- 0 处诡异事件
- 0 处 jumpscare
- 这是情感终点。守者背对、转头、合上相机，没有任何不安干扰

## 全片环境层（散布的被跟踪感）

不挂在任何具体段位，全程偶发。控制密度——玩家不应该一直 paranoid。

- **风声混入低语**：风声里偶发听不清的人哼唱 / 远方的"诶"或"啊"，每段 1-2 次，音量极低
- **雪地脚印短现**：玩家偶尔走过的地方，回头看一眼有几个不属于自己的脚印，再回头消失（只在段 4-5 雪线区域）
- **远处咳嗽 / 叹气**：玩家在某段静止超过 30s 后，远处偶发一声咳嗽或叹气

## 程序基础设施清单

W2 单点拉通先做段 1，证明系统咬合：

### 核心系统（v2 收紧 + 调味决议合并）

- [ ] `LinearLevelController`：段位状态机（trailhead / 段 1-6 / 顶峰 / 相册结尾）
- [ ] `SegmentTrigger`：进入段位触发笔记本浮现 + 解锁该段相机参数旋钮
- [ ] `ShadowEntity`：影 GameObject 通用脚本（显形条件 / 行为 / 拍照后反应）
- [ ] `PhotoEvaluator`：拍照判定（5 维评分 + passingScore 比较）
- [ ] `NotebookUI`：朋友笔记本翻页 UI（形态待定）
- [ ] `AlbumPlayback`：山顶相册自动回放本次拍的所有 PNG
- [ ] `CameraParamGate`：动态收紧 / 解锁 `CAMCOLCameraSettings` 三个 limits 字段
- [ ] `CAMExposureAccumulator` 接入 `CAMPhotoCapture` 拍照管线（段 5 星轨需要）
- [ ] 拍照存盘：现有 `CAMPhotoCapture` 已写 PNG 到 `<ProjectRoot>/photo/`，确认每张都进

### 诡异调味系统（SCP 层，详见 [`../45-uncanny.md`](../45-uncanny.md)）

- [ ] `UncannyEvent`（基类）：携带触发条件 / 概率 / 一次性 vs 重复 / 视觉 + 音效 hooks
- [ ] `UncannyTrigger`：玩家视野 / 举相机 / 段位范围 / 静止时长综合检测
- [ ] `ObservationReverseObject`（段 1 林祠用）：举相机时禁用 / 不举时启用，Layer 切换实现
- [ ] `JumpscareTrigger`（仅段 4）：一次性触发器，监听 Boss A 首次半按对焦命中
- [ ] `AmbientStalker`：环境层散布（风声混入人语 / 远处咳嗽 / 雪地脚印短现）
- [ ] `FakePhotoInjector`：拍照存盘时按概率注入预制 PNG（trailhead + 段 5 用，美术资产 W4 准备）
- [ ] `CrossSegmentAudio`（段 5 林祠低语用）：跨段位音源叠加层
- [ ] `RepositionedShadow`（段 4 雪线第二剪影 / 段 2 雾里静立人形）：玩家视野外切换位置 / 玩家走近时禁用
