# 20 · 办公室 / Hub

> 这一卷：办公室空间 + 进度系统。关卡之间的一切都在这发生。
> 关卡本身见 `30-level-design.md`。

## 空间设定

3D 第一人称可走动的办公室。规模不大，但**每件道具都带信息**：

| 区域 | 交互 | 内容 |
|---|---|---|
| 委托板（墙/桌） | 靠近 E 键 → 选关 UI | 当前可接的关卡列表 |
| 相机工作台 | 升级相机 / 换胶卷 / 换镜头 | 消耗解锁条件（见下方） |
| 线索墙（软木板/照片墙） | 查看 | 已拍到的 Boss 照、氛围照、剧情碎片，手动拖动重排 |
| 图鉴书（翻页书 / 平板） | 翻页 | 每只拍过的鬼一页，带民俗考据 |
| 档案柜 | 查看已关委托的照片存档 | 带日期、时间 |
| （可选）留声机 / 收音机 | 切换氛围音乐 | |
| （可选）窗外 | 外景随章节变 —— 四季 / 天气 | 叙事暗示 |

**TBD**：办公室是否有多房间？建议**单间开始**，需要时再开密室 / 阁楼。

## 核心循环在办公室的步骤

1. 进入办公室（通关 / 启动游戏）
2. **自动事件**（若有）：新委托信、剧情来电、线索自动上墙
3. 玩家自由探索 / 整理线索 / 升级装备
4. 靠近委托板选关 → 黑屏转场到关卡
5. 关卡结束 → 回到办公室，Boss 照上墙、图鉴自动翻到新条目

## 进度解锁轨道

随通关推进扩张。数值表见 `10-camera-rules.md § 解锁策略`。

### 相机机身（机身决定 ISO / 测光模式上限）

| 等级 | 获取方式 | 解锁内容 |
|---|---|---|
| 入门机 | 起始 | ISO ≤ 800，log-average 测光 |
| 机身 II | 通关序章 | ISO ≤ 1600，评价测光（8×5 分区） |
| 机身 III | 通关第一章 | ISO ≤ 3200 |
| 机身 IV | 通关终章前置 | ISO ≤ 6400，**对焦距离显示**、**直方图 HUD** |

### 镜头（镜头决定光圈 / 焦距）

| 镜头 | 光圈范围 | 特性 |
|---|---|---|
| 套头 | 2.8 – 16 | 普通 |
| 定焦人像 | 1.4 – 8 | 大光圈 → 暗场景友好 / 浅景深 |
| 广角 | 4 – 22 | 屋内取景容易 |
| 长焦 | 5.6 – 32 | 远距离拍怕近的鬼 |

### 胶卷 / 传感器调色（可选系统，TBD）

控制噪点脾气 / 色彩偏好。如果不做就把这栏删掉。

| 胶卷 | 噪点特征 | 色调 |
|---|---|---|
| 日常 | 中性 | 平衡 |
| 低速日光 | 噪点阈值上调 30% | 偏暖 |
| 高速夜景 | 噪点不变但暗部响应强 | 偏青冷 |

### 配件 / 道具（可选）

| 道具 | 作用 |
|---|---|
| 三脚架 | 拍照时相机模糊 × 0.2（补偿慢快门） |
| 闪光灯 | 瞬间提亮主体；**对部分鬼有惊退效果 / 对部分鬼失效** |
| 护符 / 朱砂（民俗风格） | 降低扰乱鬼干扰强度 |

### 鬼图鉴

- 每只成功拍过的鬼 → 自动填一页
- 页面内容：Boss 照、民族 / 传说出处、面具描述、**机械数值**（建议快门 / 光圈 / ISO 区间），供玩家二刷参考
- 扰乱鬼只要拍到一张就解锁；Boss 必须达标

### 线索墙

- Boss 照片强制上墙（按委托顺序排列）
- 氛围照（玩家主动拍的）可拖动自由贴
- **可选隐藏机制**：某些照片组合（如三张同一地点、三只同民族的鬼）触发新委托或剧情

## 选关 UI 结构建议

从委托板唤起，不离开 3D 空间（不要弹全屏菜单，破坏沉浸）。

```
[ 委托板 UI ]
┌──────────────────────────────┐
│ ● 当前章：第一章 · 乡镇       │
│                              │
│  01 · 古巷夜啼   [已通关 ★☆]  │  ← 可重玩
│  02 · 戏班旧影   [新]         │  ← 当前可接
│  03 · ???        [锁]         │  ← 前置未完成
│                              │
│  → 接受委托                   │
└──────────────────────────────┘
```

星标 = 额外成就（拍到所有扰乱鬼 / 在更低 ISO 下通关 / 拍到隐藏氛围物等）。
