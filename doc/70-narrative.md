# 70 · 叙事（v2）

> 这一卷：v2 的叙事骨架——主角、朋友、山、笔记本、结局。
> **2026-05-01 v2 收紧**：v1 驱魔师 + 多章节 + 委托板叙事已全部废止。

## 一句话

主角带着已故朋友的相机，沿朋友生前最后没走完的山路上山，山顶替朋友拍下他想看的画面。

## 主角

- **身份**：女孩，名字 TBD（建议 2-3 字短促）
- **年龄**：20 岁出头
- **背景**：摄影是朋友教她的，她自己不太会用相机——朋友会说"光圈先放着别动，专心调快门"，她记不全。
- **动机**：朋友登这座山没回来。她答应过他要拍下山顶的云海。这是她第一次也是唯一一次为他做这件事。
- **第一人称**：游戏里看不到她的脸。她的存在通过：脚步声 / 偶尔的内心独白（极少，关键节点 1-2 句）/ 翻笔记本时的指尖 / 举相机时的呼吸。

### 主角内心独白（候选节点 + 文案 TBD）

| 节点 | 触发 | 文案方向 |
|---|---|---|
| 开场 | 捡起相机 + 笔记本 | 一句承诺感的话："他说山顶能看到云海。我答应过他要拍下来。" |
| 段 2 雾带 | 第一次看见雾里残影 | 困惑感："这是什么？……他笔记里说要长曝。" |
| 段 4 风口 Boss A 完 | 拍下风口山君 / 没拍下 | 释然感（达标）/ 接受感（拍糊） |
| 段 6 山顶 Boss B 触发前 | 看到守者背影 | 安静的一句话："他说就是这个画面。" |
| 山顶相册回放结束 | 黑屏前 | 收尾："谢谢你。" |

> 文案具体内容由 W4 抛光阶段写定。这里只锁节点和方向。

## 朋友

- **身份**：已故，名字 TBD（结尾"献给 ___ "字幕用这个名字）
- **性别**：建议**和主角不同**（男性朋友），但不强制——更通用的处理是"亲密的朋友"，性别留模糊
- **关系**：和主角的关系不明说——可以是恋人 / 挚友 / 哥哥 / 父亲。游戏只用"朋友"代称，让玩家自己投射
- **死因**：登山事故。**不细节描述**——细节会拉低治愈基调（变成调查向）。只在段 4 风口的笔记本页里暗示"风太大了"
- **职业**：登山摄影师。这座山是他最后一次出行的目标
- **笔记本**：他生前用的旅行笔记本，6 页对应 6 段山路。**字迹手写**——美术资产手画一遍扫成 PNG，建议用钢笔字 / 中性笔字 / 略潦草。每页含：
  - 朋友的字（80-200 字）
  - 朋友当年的样片 1 张（cel-shaded 风，主角拍达标后解锁完整版，拍糊只解锁缺角版）
  - 朋友的便签 / 涂鸦 / 折角

### 朋友的"声音"调性参考

- 是枝裕和电影里的兄长 / 父亲：克制、温柔、笑着说重要的话
- 不写：戏剧化告别 / 浓烈情感 / 长篇大论
- 写："注意手不能抖——我用石头当三脚架。" 这种实用 + 暖的句子

## 山

- **名字**：虚构山名，TBD（建议中性诗意 2-3 字：白寂山 / 望舒岭 / 落山 / 长眠山 / 含山 / 默山）
- **真实地理参考**：避开任何真实神山（峨眉、华山、藏区雪山等）以免文化负担；可以参考阿尔卑斯 / 北欧山地 / 喜马拉雅外围的视觉感
- **气候**：登山过程贯穿 1 天周期：trailhead 清晨 → 段 1-3 上午 / 中午 → 段 4 风口 黄昏 → 段 5 星脊 夜 → 段 6 山顶 拂晓
- **山的"记忆"机制**：山记得每个走过的人。相机举起时，"影"显形——他们曾经走过这里、停过这里、没下来过这里。**主角的朋友也是这些影之一吗？**——这个问题留给玩家自己解读，**不明说**

## 结局

### 完整结局（Boss B passingScore ≥ 70）

1. 山顶守者拍达标 → 守者缓缓转头
2. 主角合上相机
3. 黑屏 → 相册回放本次拍的所有 PNG（含 Boss B 达标照定格 4 秒）
4. 黑屏字幕 "献给 ___"
5. 朋友最后一封信文案出现（TBD，建议 100-150 字，他写给主角的最后一段话）
6. 字幕滚屏 → 主菜单（重玩 / 退出）

### 中性结局（Boss B 拍糊 / 没拍）

1. 守者保持背对，云海一片白光罩住画面
2. 主角合上相机
3. 黑屏 → 相册回放本次拍的所有 PNG
4. 一张白光照定格
5. 黑屏字幕 "献给 ___"
6. 字幕滚屏 → 主菜单

**两个结局都 OK**。中性结局**不是失败**——是另一种"看到了"。这是 v2 治愈基调的核心：尊重技能不到位的玩家也能完整体验情感线。

### 隐藏奖励层（不分支结局，但加内容）

- Boss A 风口山君拍达标 → 笔记本隐藏页 "我也没拍成。谢谢。" + 朋友的一张山顶手记照
- 4 个 minion 全部拍达标 → 相册回放时多 4 张朋友当年拍过的同位置对比图（拍出"代友走完"的视觉对仗）

## 不写什么

- ❌ 不写恐怖元素：jump scare / 鬼追击 / 死亡 / 失踪 / 调查 / 真相反转
- ❌ 不写大量叙事文本：避免主角内心独白长段；避免全屏字幕 / 章节标题 / 章节过场
- ❌ 不写战斗 / 选择题 / 多结局分支
- ❌ 不解释一切：守者是谁？影是什么？朋友的死因细节？——**留模糊**
