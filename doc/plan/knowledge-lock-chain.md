# 知识锁链（v2 单关版）

> **2026-05-01 v2 收紧后**：单关 = 单条教学曲线。v1 的"L1 教快门 / L2 教 ISO / ..."多关分阶解锁全部归档。改这份文档：直接改即可（不算规则层修订），但要在 [`decisions-log.md`](decisions-log.md) 里记一条。

## 总览

游戏开始时玩家**只能用最基础的相机操作**（举起 / 半按对焦 / 全按拍照）。其余参数随登山路径推进**通过朋友的笔记本翻页解锁**——叙事即教学。

| 段 | 路径地形 | 笔记本翻页解锁 | 教什么 | 对应影 |
|---|---|---|---|---|
| 0 | 主菜单 → trailhead | （初始）举相机 + 半按对焦 + 全按拍照 | 基础操作 | （无） |
| 1 | 林线 trailhead | **第 1 页**：构图取景框（黄金分割辅助线）| 半按对焦 + 中央构图 | 林祠静坐者 |
| 2 | 雾带缓坡 | **第 2 页**：慢快门刻度（1/30 → 1/2 → 1s）+ 朋友的样片"1.2s 才显形" | 长曝消形 | 雾里残影群 |
| 3 | 岩壁 | **第 3 页**：快快门刻度（1/250 → 1/1000）+ ISO 100-400 | 快门冻结 + ISO 基础 | 山鸦掠影 |
| 4 | 风口 / 雪线 | **第 4 页**：ISO 800-1600 + 噪点说明 + 朋友的失败照片"风太大没拍清" | 综合：快门 × ISO 噪点 | **Boss A · 风口山君** |
| 5 | 山脊夜段 | **第 5 页**：1s 以上长曝 + 三脚架 + 星轨拍法 | 长曝积累 / 星轨 | 星灵 |
| 6 | 顶峰 | **第 6 页**：手动对焦 / 大光圈 / 景深 + 朋友的最后一段话"我想看的是这个" | 景深 + 对焦距离 + 长曝综合 | **Boss B · 山顶守者** |

每翻一页 = 玩家相机的某个参数旋钮**从锁定变可调**。这不是 UI 弹窗教学，是**笔记本里手写的字 + 朋友拍的样片**让玩家"看懂"该怎么用。

## 解锁的参数边界（单关全开）

| 参数 | 范围 | 解锁时机 |
|---|---|---|
| 快门 | **8s – 1/2000** | 段 0 = 1/250-1/60 锁；段 2 解锁慢档；段 3 解锁快档；段 5 解锁 8s 长曝 |
| ISO | **100 – 3200** | 段 0 = 100-400 锁；段 3 解锁 800；段 4 解锁 1600-3200 |
| 光圈 | **f/1.4 – f/16** | 段 0-5 = f/4 固定；段 6 解锁全光圈范围 |
| 测光模式 | 中央重点（log-average） | 全程 |
| 对焦 | 自动半按 / 段 6 解锁手动 | 段 6 解锁手动对焦距离调节 |
| 闪光 | 不解锁 | v2 不做 |
| 曝光补偿 | -2 EV — +2 EV | 全程 |

## 教学目标（玩家通关时必须真的"懂"）

1. **半按对焦 + 取景构图**：把影放在画面中央 70% 区域 + 焦在影身上
2. **慢快门长曝消形**：≥ 1s 曝光会让持续移动的雾形/残影擦除，露出后面的真相
3. **快快门冻结**：高速运动（山鸦 / Boss A 冲风）需要 1/500+ 才不糊
4. **ISO 噪点权衡**：暗光下被迫升 ISO，但 ≥ 1600 噪点显著；玩家自己决定降快门 vs 升 ISO
5. **长曝星轨**：星空 + 1s+ 曝光产生轨迹效果（机制接 `CAMExposureAccumulator`）
6. **景深 / 大光圈虚化**：手动对焦距离 + 大光圈 → 主体清晰背景虚化（Boss B 山顶守者考察这点）

## passingScore 一览

| 影 | passingScore | 备注 |
|---|---|---|
| 林祠静坐者 | 50 | 破冰，门槛低 |
| 雾里残影群 | 60 | 必须在长曝下显形 |
| 山鸦掠影 | 60 | 横向高速，1/500 才能冻结 |
| **Boss A · 风口山君** | 65 | 综合考；拍达标 = 隐藏笔记本页 |
| 星灵 | 60 | 必须在长曝积累下出星轨 |
| **Boss B · 山顶守者** | 70 | 综合大考；拍达标 = 完整结局 |

**注**：拍糊（< passingScore）**不会卡通关**，影短暂淡出再显形，玩家可无限尝试。所有影都不会主动伤害玩家。
