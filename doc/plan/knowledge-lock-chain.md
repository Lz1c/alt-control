## L1 · 快门主题（候选锁，待 dry-run 后正式锁定）

> **2026-04-25 改名**：原"快门优先"暗示只教"拉快门冻结快速运动"——这只是快门的一个属性。改成"**快门主题**"，单关讲透三种快门哲学：冻结、长曝（消形/积累）、摇镜（多形态显影）。详见 `decisions-log.md` 同日"L1 升级为快门主题关 (B+C 路线)"条目。

### 教学目标

玩家在通关 L1 之后，必须真的"懂"快门**作为一种创作工具**：

- **冻结**：快门越快越凝固高速运动（基础直觉）
- **长曝消形**：≥ 1s 曝光 + 防抖 → 持续移动的物体被擦除（反直觉）
- **长曝积累**：长曝可以让"动作过程"留下轨迹（反直觉）
- **摇镜法**：跟随主体 + 慢快门 → 主体清晰 + 背景流线（表达速度感）
- **多形态显影**（Boss 专属）：不同快门档位下，鬼显示的真相不一样——慢快门字面看到更多
- ISO 和光圈在这一关被锁住，只能调快门，强迫玩家把"快门"这一个变量的多面学透

### 解锁参数（这一关玩家相机能调到的范围）

| 参数 | 范围 | 备注 |
|---|---|---|
| 快门 | **1s – 1/1000**（含半档；含 1s / 1/2 / 1/4 / 1/8 / 1/15 慢档位） | 主教学目标。慢档为长曝消形/积累必需；快档为冻结必需 |
| ISO | 100 / 200 / 400 / 800 四档 | 上限 800 是 Boss 冲刺 + 冲跑小孩的必需值；同时是 L2 ISO 教学的伏笔 |
| 光圈 | 固定 f/4 | 这一关不教光圈 |
| 测光模式 | 中央重点 | `CAMMetering`（log-average 那个），不暴露 evaluative |
| 对焦 | 自动（半按对焦） | 不暴露手动对焦距离调节 |
| 闪光 | 未解锁 | 留给 L5+ |
| 曝光补偿 | -1 EV — +1 EV | 给一点弹性，但不是核心 |

## L2..N（候选方向，不承诺）

### L2 候选 · ISO + 暗光噪点

- 教学：暗光下必须提高 ISO，但 ISO ≥ `isoNoiseThreshold` 后照片噪点会被 PhotoEvaluator 扣分
- 候选鬼：`ghost-xianying`（只在 PNG 显形）或 `ghost-xiaomian`（对焦触发）
- 新解锁：ISO 800-1600

### L3 候选 · 对焦距离 / 景深

- 教学：手动对焦 + 大光圈虚化 + 最低对焦距离
- 候选鬼：`ghost-beimian`（只能从反射面拍）
- 新解锁：手动对焦 + 光圈 f/2 – f/8

### L4 候选 · 构图 + 分身

- 教学：PhotoEvaluator 加"唯一性"维度（拍到几只分身要算）
- 候选鬼：`ghost-fenshen`
- 新解锁：评分系统增项

### L5+ · 闪光 / 长曝（远期）

- 大概率涉及代码改动（`CAMExposureAccumulator` 接入拍照链路）
- 留到 L4 验证后再设计
