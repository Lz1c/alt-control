> **2026-05-01 v2 收紧后版本。** 一个月交作业，单关登山治愈。v1（驱魔师 + 多关 + 傩戏面具）已归档，详见 [`plan/decisions-log.md` 2026-05-01](plan/decisions-log.md) 与 [`plan/rules-revisions.md`](plan/rules-revisions.md)。

## 一句话

玩家是一个**不太会用相机的女孩**，带着已故朋友留下的相机，沿朋友生前最后没走完的山路上山。一路上**只有透过取景器才能看见山记得的人和事**——不是鬼，是山的记忆。山顶替朋友拍下他想看的那张照片。

## 设计支柱

1. **相机即视觉**。山的记忆对肉眼不可见；只有透过相机取景器才能显形。"举起相机"不是装饰动作，而是玩家**进入"能看见"状态**的开关。
2. **相机即语言**。玩家的全部表达都压在相机参数上 —— ISO / 快门 / 光圈 / 对焦 / 测光，全部硬核、全部关联。但**主角不会用，玩家也不会用**——两者一起学。
3. **教学即叙事**。教学 = **朋友留下的笔记本**。每段山路对应笔记本的一页：朋友亲笔写的相机参数小抄、当年的照片样片、未完成的备忘。玩家学相机 = 主角破译朋友的笔记 = 替朋友走一遍。**没有干巴巴的 UI tooltip**。
4. **影 = 山的记忆**。山上的"影 / 余像"不是恐怖的鬼，是过去走过这座山的人、动物、瞬间——被山保存下来。**相机让它们短暂显形**。每个影对应一段相机概念的考察。
5. **不卡通关，达标 = 完整结局**。PhotoEvaluator 出分，**但拍糊不会卡死**——拍糊 = 中性结局"看到了但没拍下来"；拍达标 = 完整结局"替朋友拍下了那张画面"。治愈基调 + 作业级体量不在技能门槛上设关。
6. **照片是真的**。照片是经过相机模拟管线输出的真 PNG（见 `10-camera-rules.md`），保存到磁盘。**山顶玩家会回顾自己一路拍的所有照片** —— 这是替代收集墙的情感收尾。
7. **第一人称纯走路**。不做攀岩。Cairn 给场景和移动手感，不给攀岩。

## 核心循环（单关 = 整个游戏）

```
开场（trailhead）
   │
   ├─ 相机包里掉出朋友的笔记本 + 朋友的相机
   ├─ 玩家学：举起相机 / 半按对焦 / 拍照
   │
   ▼
登山（线性 6 段路径，每段教一个相机概念 + 1 只影）
   │
   ├─ 段 1 林线 trailhead → 林祠静坐者（破冰）
   ├─ 段 2 雾带缓坡 → 雾里残影群（长曝消形）
   ├─ 段 3 岩壁 → 山鸦掠影（快门冻结）
   ├─ 段 4 风口/雪线 → Boss A · 风口山君（综合：快门 × ISO）
   ├─ 段 5 山脊夜段 → 星灵（长曝积累 / 星轨）
   ├─ 段 6 顶峰 → Boss B · 山顶守者（综合：景深 × 对焦 × 长曝）
   │
   ▼
山顶
   │
   ├─ Boss B 拍下：达标 = 完整结局 / 拍糊 = 中性结局
   ├─ 主角合上朋友的相机
   │
   ▼
相册回放（结尾）
   │
   ├─ 自动翻看玩家一路拍的所有 PNG
   ├─ 黑屏 → 一行字"献给 ___"（朋友的名字）
   └─ 制作人员 / 重玩按钮
```

## 玩家目标层级

| 层 | 目标 | 反馈 |
|---|---|---|
| 必做 | 登顶 | 看到山顶画面 / 中性结局 |
| 推荐 | 拍达标 Boss B | 完整结局 + 相册彩页 |
| 选做 | 沿途拍下 4 个 minion 影 | 相册更厚、笔记本翻得更全 |
| 隐藏 | 拍达标 Boss A（风口山君） | 朋友笔记本里多一页"我也没拍成，谢谢" |

## 拍照判定原则（摘要）

详细见 `10-camera-rules.md § 拍照判定`。v2 核心规则：

- **必要条件**：影出现在画面中央区域内、对焦在影身上。
- **达标阈值**：每只影有 `passingScore`（基线 60，Boss B 70）。
- **拍糊不惩罚**：影**短暂淡出 + 重新显形**（约 3-5 秒），玩家继续尝试。无胶卷消耗、无追击、无失败状态。
- **达标 vs 不达标**：达标解锁影对应的笔记本页"完整版"（朋友当年的样片 + 一段文字）；不达标只解锁"半页"（影像缺角）。

## 主角 / 叙事骨架

- **主角**：女孩，名字 TBD（建议短促易记，如"阿璃 / 林晚 / 阿青"）。第一人称视角，**不出现镜头里**——你看到的世界就是她看到的。
- **朋友**：已故，名字 TBD。生前是登山摄影师，这座山是她最后一次出行。**笔记本里全是她的字迹**。
- **山**：虚构山名（建议中性诗意，如"白寂山 / 望舒岭 / 落山"），避开任何真实神山以免文化负担。
- **基调**：克制、冷峻、温柔。Cairn 风的沉郁基底配主角内心独白（极少，关键节点 1-2 句）。
- **不做**：吓人 jump scare / 战斗 / NPC 对话 / 多结局分支（只 1 个完整 + 1 个中性）。

## 不做什么（v2 反设计）

- **不做战斗 / 不打影**。相机是唯一"语言"，拍到就是表达。
- **不做开放世界 / 多关卡 / 选关 UI**。游戏从主菜单 → 直接进 trailhead。
- **不做面具收集 / 图鉴 / 办公室 hub**。新收集物 = 玩家相册（自动存盘）。
- **不做攀岩 / QTE / 跳跃挑战**。第一人称走路 + 拍照。
- **不做自动曝光替玩家搞定**。半按测光是参考读数，玩家自己看场景。
- **不做难度档**。治愈基调下不设技能门槛。
- **不做 jump scare / 鬼追击**。"影"是温柔的，**不会主动伤害玩家**。
