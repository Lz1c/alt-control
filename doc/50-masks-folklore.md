# 50 · 面具 / 民俗

> 这一卷：傩戏面具 + 少数民族民俗的**设计规则与模板**。调研素材不在本卷。
>
> **完整调研**：[`90-reference/nuo-masks.md`](90-reference/nuo-masks.md)（非遗名录、造型志、工艺、游戏化建议、参考文献）
>
> **态度**：借鉴、致敬、改编，不冒犯、不照搬特定宗教仪式。具体面具造型**必须是原创变体**，不直接复刻现存文物或活态仪式用物。

## 五色体系（直接用于美术 / 鬼谱）

傩面具在千年演化中形成了稳定的象征色谱，**整个游戏的鬼和面具都应该落在这套色谱上**，方便玩家潜意识形成识别。

| 色 | 象征 | 典型神格 | 建议用在哪类鬼 |
|---|---|---|---|
| **红** | 血祭 / 威慑 / 忠义 | 关公、朱砂判官 | 愤怒型 / 守护型 / 含冤型 Boss |
| **黑** | 阴界 / 镇邪 / 刚直 | 周仓、无常 | 镇守型 / 审判型 |
| **白** | 亡灵 / 哀祭 / 病气 | 白无常、孝服神 | 悲情型 / 游荡型 / 病祟类扰乱鬼 |
| **青 / 蓝 / 绿** | 山林 / 妖兽 / 非人 | 开山莽将、山魈 | 兽相 Boss / 非人形扰乱鬼 |
| **金** | 神格 / 吉祥 / 不可触 | 天神、护法 | 极高阶 / 终章 Boss / 特殊奖励照 |

> **设计原则**：一只鬼的主色 = 它在叙事里"是什么"，不是随便挑的。
>
> 反例：不要把一个悲情少女 Boss 做成红黑凶神面具；她应该是白色为主、附以金饰提示"曾有神格"。

## 造型元素词表

调研里总结的傩面具"超常态特征"：**獠牙 / 三眼 / 鼓目 / 歪嘴 / 吐舌 / 犄角 / 额心瞳（铜镜嵌额）**。每只 Boss 鬼至少选 1-2 项叠加在基础人面上，这是"一眼可辨、过目难忘"的关键。

活口 / 半截面具（下颌或眼睛可动）是德江独门工艺，**非常适合 Boss 在"揭面"瞬间触发机关演出**。

## 地域 / 体系参考清单

以下是调研里按活态谱系整理的真实体系。每一个都是设计可以**落锚**的方向，选 1-2 个作为本作主风格，其余做客串 / 章节切换。

| 体系 | 地域 | 调性 | 典型面具特征 | 适合做什么 |
|---|---|---|---|---|
| **威宁撮泰吉（彝）** | 贵州威宁板底 | 最原始、祖先相 | 猿相、无眼珠、墨底白线皱纹 | 序章 / 远古 / 祖灵 Boss |
| **德江傩堂戏（土家）** | 贵州德江 | 最丰富、神人兽并立 | 24 面全堂、傩公傩母、开山獠牙、活口机关 | 主要风格基底（神谱多，够一整个游戏用） |
| **安顺地戏（汉军傩）** | 贵州安顺 | 最华丽、盔面连体 | 头盔 + 大面 + 耳翅、雉尾、红黑蓝白程式 | 武将 / 镇守型 / 高华丽度关卡 |
| **辰州傩（湘西）** | 沅陵七甲坪 | 精致三分 | 正神 / 凶神 / 世俗三类分明 | 叙事层次感强的章节 |
| **仡佬族活口面具** | 贵州道真 | 机械诗意 | 可活动下颌 / 眼球 | Boss 揭面机关戏 |
| **荔波布依傩** | 贵州荔波 | 秀雅清丽 | 淡彩、俊美、不狞厉 | 对抗"东方 = 狰狞"刻板印象的调节项 |
| **侗族咚咚推** | 湖南新晃 | 36 交目 | 朴拙、黑白对比强 | 小民族专题章节 |
| **毛南族肥套** | 广西环江 | 狞厉与温柔双极 | 36 神面，万岁娘娘慈祥 + 凶神对比 | 祈子 / 送子 / 家族线剧情 |
| **壮族师公 36 神** | 广西 | 神职官僚体系 | 三元真君 + 三十六神 | 可做"卡池"式图鉴结构 |
| **关索戏（云南军傩）** | 澄江小屯村 | 纸胎漆布 | 三国关羽父子主题 | 汉英雄神化 / 历史钩 |
| **藏羌姆 / 藏戏** | 藏区 | 最奢华 / 蓝白两派 | 忿怒相护法、金色冠冕 | ⚠️ **慎用**，宗教敏感度最高 |

**哪些优先做**（个人建议，可改）：

- **主风格基底**：德江傩堂戏（神谱最全）
- **序章味道**：威宁撮泰吉（原始祖灵感）
- **差异化调节**：荔波布依（证明游戏里不是只有狞厉）
- **机关 Boss**：仡佬活口
- **暂不涉入**：藏传羌姆（宗教风险太高）

## 通用创作准则

1. **先查原型，再改造**。每只鬼的面具都要能说出"参考自什么" —— 但最终美术资产是**原创设计**。
2. **不复刻活态仪式用物**。许多坛班仍在举行真实还愿 / 冲傩；把他们的真实祭祀物做成娱乐装备是冒犯。
3. **不使用真实符咒原文与完整仪式动作链**。可以用仪式**结构**（开坛—请神—酬神—送神），不用仪式**内容**。
4. **主色与地域绑定**。参见上方五色体系；一个关卡 / 章节尽量统一色板，让玩家潜移默化记住。
5. **面具 ≠ 鬼**。"戴上面具是神，摘下面具是人"是傩的核心民俗逻辑。照片打穿幻象时可能看到面具下是另一张脸（剧情钩子）。
6. **同一体系的不同鬼**共享工艺（木/漆/银/纸/盔）和色板，让图鉴成系列。
7. **保留雅俗并置**。不要把全部面具都做成狰狞 Boss —— 调研里德江有秦童歪嘴丑角、地盘和尚无下巴、梁九清官诙谐。世俗幽默面具可以做成扰乱鬼或氛围物，反差更强。

## 三大风险（必读）

调研里点名的游戏化傩戏最容易翻车的三个地方：

1. **文化挪用**：傩是活态非遗。建议 —— 每个面具条目都注明参考来源（民族 / 坛口 / 非遗编号）；涉及真实传承人时标注"致谢"而不是"参考"。若有条件，做一次田野并联系一位传承人。
2. **刻板东方主义**。傩不是"神秘东方 = 狞厉恐怖"的装饰标签。荔波布依和毛南万岁娘娘证明傩的美学光谱远比"恐怖"丰富；游戏里至少要有一只"慈眉善目"的 Boss 或 NPC，避免整体调性一根筋。
3. **过度神秘化 / 宗教冒犯**。藏传佛教忿怒相护法（玛哈嘎拉、阎魔敌等）**不要**直接做成反派。可以借用视觉语言但不要挪用神名和身份。

## 面具卡模板

复制到下方做新条目。

```markdown
### mask-xxx · 面具名

**关联鬼**：`ghost-xxx`
**体系参考**：德江傩堂戏 / 威宁撮泰吉 / …（必填，对应上面清单之一，或写"原创混合：A + B"）
**仪式语境**（原型大致用途）：驱疫 / 祭祖 / 还愿 / 祈子 / 丧葬超度 / ...
**主色**：红 / 黑 / 白 / 青 / 金（对应五色体系）
**材质**：木雕 + 漆 / 纸扎 + 布 / 皮革 / 盔面连体 / 活口机关 / ...
**造型元素**：从"獠牙 / 三眼 / 鼓目 / 歪嘴 / 吐舌 / 犄角 / 额心瞳"中选 1-2 项
**原创变形**：在原型上改了什么（必填。例：原型对称，游戏里左右半脸不同）
**象征含义**（游戏叙事层）：这只鬼为什么戴这面具
**美术参考**：参考图 URL 或存档路径
**致谢 / 来源**：传承人 / 地域 / 非遗编号（如适用）
```

## 面具编号命名

`mask-<体系>-<序号>`。例：

- `mask-dejiang-01` ~：德江傩堂戏系
- `mask-cuotaiji-01` ~：威宁撮泰吉系
- `mask-dixi-01` ~：安顺地戏系
- `mask-maonan-01` ~：毛南肥套系
- `mask-han-01` ~：汉地民俗（钟馗 / 无常 / 判官）
