# 50 · 面具 / 民俗

> 这一卷：傩戏面具 + 少数民族民俗的设定考据与视觉参考。
> **态度**：借鉴、致敬、改编，不冒犯、不照搬特定宗教仪式。具体面具造型建议原创变体，不直接用现存文物。

## 面具卡模板

复制到下方做新条目。

```markdown
### mask-xxx · 面具名

**关联鬼**：`ghost-xxx`
**民族 / 地域参考**：
**仪式语境**（原型大致用途）：驱疫 / 祭祖 / 丰收 / 婚丧 / ...
**主色 / 材质**：木雕 + 漆 / 纸扎 + 布 / 皮革 / ...
**显著造型特征**：角 / 獠牙 / 面纹 / 贴金 / ...
**原创变形**：在原型上改了什么（例：原型对称，游戏里左右半脸不同）
**象征含义**（游戏叙事层）：这只鬼为什么戴这面具
**美术参考**：参考图 URL 或存档路径
```

## 通用创作准则

1. **先查原型，再改造**。每只鬼的面具都要能说出"参考自什么" —— 但最终美术资产是原创设计。
2. **不使用现存宗教活态仪式的完整动作链**。避免让 NPC 表演某个可识别的真实仪式流程。
3. **主色与地域绑定**。红黑 = 傩 / 火把；银白 = 苗；土黄木色 = 木雕；纸白 = 纸扎。玩家潜移默化记住。
4. **面具 ≠ 鬼**。面具是"鬼戴的"，不是"鬼本身"。照片打穿幻象时可能看到面具下是另一张脸（剧情钩子）。
5. **同一民族的不同鬼**共享一套色板和工艺（木/漆/银/纸），让图鉴按章节看起来成系列。
