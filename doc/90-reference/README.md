## 加入新调研时

1. 原文放这里，文件名 slug 化（`cairn-art-analysis.md` 而不是 `Cairn 美术分析.md`），方便链接和引用。
2. 不要改原文（除非是修正明显错字）。若要做脚注 / 批注，在文件顶部加一段"编辑记录"或插 `> [NOTE]` 引用块，不要把原文拆散。
3. 在对应的策划案卷里加一行 `> 完整调研见 [90-reference/xxx.md](90-reference/xxx.md)`，并把**设计可用**的要点提炼一段放进策划案——策划案永远是"已加工过的结论"，读者不必看调研原文也能用。
4. 回到本 README 更新上表。
