> 这一卷：**v2 唯一一关**（一座山）的结构骨架。具体内容写在 [`60-levels/01-shan.md`](60-levels/01-shan.md)。
>

## 关卡 = 一座山

游戏只有一关，关卡 = 整个游戏。线性 6 段山路 + 1 个开场 trailhead + 1 个结尾相册回放。

```
主菜单
   │
   ▼
trailhead 开场（朋友相机 + 笔记本入手）
   │
   ▼
段 1 林线 ─→ 段 2 雾带 ─→ 段 3 岩壁 ─→ 段 4 风口（Boss A）
                                                       │
                                                       ▼
段 6 顶峰（Boss B） ←─ 段 5 星脊
   │
   ▼
山顶相册回放 → 黑屏 → "献给 ___"
```

**不可逆**。玩家只能向上走（用地形挡板做软引导）。回头看可以，但不能掉头跑回 trailhead。

## 关卡的组成

| 层 | v2 内容 | 是否必需 |
|---|---|---|
| **地块** | Cairn 借用的山系场景，6 段地形拼接 | 必 |
| **影（Boss + minion）** | 4 minion + 2 Boss = 6 只，每段 1 只 | 必 |
| **笔记本节点** | 段位入口处自动浮现朋友的笔记本一页（教学）| 必 |
| **氛围异象** | 朋友走过的痕迹（旧三脚架印 / 营地余烬 / 岩石上的便签 / 早年照片）| 必（叙事）|
| **拍照点位** | 每段 1 个"建议拍点"（朋友当年拍过的位置标记）+ 若干自由拍点 | 必 |

## 段位生命周期

每段山路都用同一个模板：

```
进入该段（地形过渡 / 视野改变 / 笔记本自动浮现）
   │
   ▼
朋友的笔记本页：教这一段需要的相机概念
   │  + 朋友当年的样片（如果有）
   │  + 朋友的便签 / 一句话
   │
   ▼
玩家走到该段的"建议拍点"（视野最好的位置）
   │
   ▼
举起相机 → 看到本段的影显形
   │
   ▼
拍照
   │
   ├─ 达标（≥ passingScore）→ 笔记本完整版解锁 + 影留下"印记"（影点位旁出现一道光/发光纹饰）
   ├─ 拍糊（< passingScore）→ 笔记本半页 + 影短暂淡出（3-5 秒）后重新显形，玩家可继续尝试
   └─ 跳过（玩家走过去不拍）→ 笔记本仍开半页提示"这里你略过了"，玩家可以回头补拍（在该段范围内）
   │
   ▼
继续向上走 → 进入下一段
```

## Boss 段（段 4 风口 / 段 6 顶峰）

Boss 段比 minion 段多两层：

- **环境强化**：风口段有粒子风暴 + 暗光；顶峰段有云海 + 极远景
- **passingScore 更高**：minion 50-60，Boss A 65，Boss B 70
- **隐藏页奖励**：Boss A 拍达标 → 笔记本多一页"我也没拍成，谢谢"；Boss B 拍达标 → 完整结局

**Boss 不阻挡推进**。即使玩家拍糊或跳过 Boss A，仍可继续向上。Boss B 必须遭遇（位于山顶，物理上必经之地），但拍糊也能进入相册回放（中性结局）。

## 失败策略（v2 治愈基调）

**没有失败状态**。

| 玩家行为 | 反应 |
|---|---|
| 拍糊 | 影短暂淡出 3-5 秒后重新显形；笔记本只解锁半页 |
| 跳过 | 笔记本半页提示，可在该段范围内回头补拍 |
| 来回走 | 不限制（除了不能掉头下山）|
| 关游戏 | 重新开始时从 trailhead 重新走（v2 不做存档续玩）|

**没有**：胶卷消耗 / Game Over 屏 / 重试 prompt / 影追击 / 死亡 / 跌落。

## 通关条件

- **必要**：登顶（物理走到山顶 Boss B 段）+ 至少举相机看了一次 Boss B（不要求拍达标）
- **完整结局**：Boss B 拍达标（≥ 70）
- **中性结局**：Boss B 拍糊或没拍

两个结局都进相册回放，区别在结尾字幕和相册彩页数。

## 氛围物预算（参考）

| 类别 | 整关数量 | 例子 |
|---|---|---|
| 朋友的便签 | 6 段 × 1 张 = 6 张 | 笔记本翻页一并展示 |
| 朋友走过的痕迹 | 5–8 件 | 旧三脚架印、营地余烬、半截相机带、绑带、刻字岩 |
| 朋友的早年照片 | 6 段 × 1 张 = 6 张样片 | 出现在笔记本对应页 |
| 环境异象 | 5–8 件 | 远处风吹云走、雪线之上的鸟群、雾里一闪即逝的身影、不属于这个时代的旧物 |

## 程序基础设施清单（整个项目）

- [ ] `LinearLevelController`：管理段位状态机（trailhead / 段 1-6 / 顶峰 / 相册结尾）

- [ ] `AlbumPlayback`：山顶相册自动回放本次拍的所有 PNG
- [ ] `CameraParamGate`：限制 `CAMCOLCameraSettings.isoLimits` 等动态收紧 / 解锁
