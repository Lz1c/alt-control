# 40 · 鬼谱 / Bestiary

> 这一卷：所有鬼的机械数值 + 设定。D&D Monster Manual 风格。
> 每只鬼有一张卡，按章节排序。新增鬼时**严格按模板填**，不要添字段不加到模板。

## 两种鬼

| 类 | 作用 | 必须挂的组件 |
|---|---|---|
| **Boss 鬼** | 关底目标，必须拍到达标照才通关 | `CAMMotionBlurSubject`、`Ghost`（TBD 脚本）、`BossMarker` |
| **扰乱鬼** | 陪衬、干扰、图鉴扩充，可拍可不拍 | `CAMMotionBlurSubject`、`Ghost` |

## 通用字段

所有鬼共享的机械字段。这些都来自 `10-camera-rules.md` 的规则，**不要在这卷里重新发明数值含义**。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | string | 唯一 ID，例如 `ghost-guniang` |
| `name_cn` / `name_en` | string | 显示名 |
| `民族 / 出处` | string | 傩戏、彝族、苗族、汉俗…… |
| `mask_id` | ref → `50-masks-folklore.md` | 面具编号 |
| `外观` | 文本 | 身高、服饰、姿态、显著特征 |
| `行为 AI` | 枚举 + 文本 | 游荡 / 追逐 / 凝视 / 瞬移 / 隐身…… |
| `移动速度` | m/s | 世界坐标速度参考 |
| `blur_multiplier` | float | `CAMMotionBlurSubject.blurMultiplier`，默认 1 |
| `最低对焦距离` | m | 低于这个距离时鬼会后退或虚化 |
| `亮度偏好` | 枚举 | 极暗 / 暗 / 中 / 亮 |
| `关联谜题` | 文本 | 这只鬼影响哪些关卡机制 |

**分类**：Boss 鬼
**首次出现关卡**：01 · 古巷夜啼

#### 设定
- 民族/出处：TBD
- 面具：`mask-tbd`
- 外观：TBD
- 叙事背景：TBD

#### 行为
- AI 模式：游荡 + 间歇凝视
- 出现模式：在巷口远端突然显形，然后缓慢接近
- 移动速度：1.2 m/s

#### 机械数值
| 项 | 值 |
|---|---|
| `blur_multiplier` | 0.8（较慢，宽容） |
| 推荐快门 | 1/60 s |
| 推荐光圈 | f/2.0 – f/4 |
| 推荐 ISO | 800 – 1600 |
| 最低对焦距离 | 2.5 m |
| 亮度偏好 | 暗 |
| 占屏比要求 | ≥ 12% |
| `passingScore` | 60 |

#### 拍照判定要点
- 必须在她"凝视玩家"的 2 秒窗口拍；她低头时拍会失败。

#### 失败反应
- 原地消失 5 秒 → 在更远处重新显形。

**分类**：扰乱鬼
**出现关卡**：01, 02

#### 设定
- 民族/出处：TBD
- 外观：只在余光 / 镜面里出现的黑影
- 叙事背景：TBD

#### 行为
- AI 模式：玩家回头即消失
- 干扰方式：一闪而过，误导玩家看错方向

#### 机械数值
| 项 | 值 |
|---|---|
| `blur_multiplier` | 3.0（极快，需很快的快门） |
| 图鉴解锁条件 | 拍到一张，"中心" + "对焦 ≤ 30% 误差" |
| 给予奖励 | 线索碎片 × 1 |

#### 设计意图
- 教玩家"有些鬼不能正面遭遇" —— 余光 / 反射里更容易捕捉。
- 同时做图鉴的高难度追击目标。
