# 40 · 鬼谱规则 / Bestiary Rules

> 这一卷：**鬼的规则、字段定义、两种鬼卡模板**。不放具体鬼的条目。
>
> **具体鬼**：[`41-ghosts/`](41-ghosts/README.md)（一鬼一文件）
>
> 新增鬼走 `/new-ghost` 斜杠命令；它会按本卷模板建文件、更新 `41-ghosts/README.md` 和 `51-masks/README.md` 的索引。

## 两种鬼

| 类 | 作用 | 必须挂的组件 |
|---|---|---|
| **Boss 鬼** | 关底目标，必须拍到达标照才通关 | `CAMMotionBlurSubject`、`Ghost`（TBD 脚本）、`BossMarker` |
| **扰乱鬼** | 陪衬、干扰、图鉴扩充，可拍可不拍 | `CAMMotionBlurSubject`、`Ghost` |

## 通用字段

所有鬼共享的机械字段。这些都来自 `10-camera-rules.md` 的规则，**不要在这卷里重新发明数值含义**。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | string | 唯一 ID，例如 `ghost-guniang` |
| `name_cn` / `name_en` | string | 显示名 |
| `民族 / 出处` | string | 傩戏、彝族、苗族、汉俗…… |
| `mask_id` | ref → `51-masks/` | 面具编号 |
| `外观` | 文本 | 身高、服饰、姿态、显著特征 |
| `行为 AI` | 枚举 + 文本 | 游荡 / 追逐 / 凝视 / 瞬移 / 隐身…… |
| `移动速度` | m/s | 世界坐标速度参考 |
| `blur_multiplier` | float | `CAMMotionBlurSubject.blurMultiplier`，默认 1 |
| `最低对焦距离` | m | 低于这个距离时鬼会后退或虚化 |
| `亮度偏好` | 枚举 | 极暗 / 暗 / 中 / 亮 |
| `关联谜题` | 文本 | 这只鬼影响哪些关卡机制 |
