# 40 · 影 / 鬼谱规则（v2）

> 这一卷：**影的规则、字段定义、两种影卡模板**。不放具体影的条目。
>
> **具体影**：[`41-ghosts/`](41-ghosts/README.md)（一影一文件）
>
> 新增影走 `/new-ghost` 斜杠命令。⚠️ 用 `/new-ghost` 时**口头明确告知 v2 治愈基调 + 山系**，否则它可能按 v1 民俗调性写。

## v2 术语（重要）

- **"影 / 余像 / 旧时人 / 山的记忆"** = v2 的鬼。设计上是温柔的、沉默的、不主动伤害玩家。**山记得每个走过的人**。
- **技术 ID 仍叫 `ghost-*`**：保留 `41-ghosts/`、`ghost-xxx.md`、`Ghost` 脚本类名等所有内部 ID，避免破坏数百处引用。**只在 prose / UI / 玩家可见文案里说"影"**。
- **不要在 v2 影卡里写**：恐怖 / 攻击 / 追击 / 伤害 / jump scare / 民俗体系 / 傩戏 / 面具 / 五色 / 民族出处。这些都是 v1 概念，全部归档。

## 两种影

| 类 | 作用 | 必须挂的组件 |
|---|---|---|
| **Boss 影** | 关键节点显形（v2 = 风口段 + 顶峰段两只），拍达标解锁完整笔记本 / 完整结局 | `CAMMotionBlurSubject`、`ShadowEntity`、`BossMarker` |
| **扰乱影**（minion） | 沿途显形，每只对应一段相机概念教学 | `CAMMotionBlurSubject`、`ShadowEntity` |

**v2 与 v1 的区别**：扰乱影**全部不主动伤害玩家**；不再分"温和型 / 攻击型"。所有影都温和。

## 通用字段

所有影共享的机械字段。这些都来自 `10-camera-rules.md` 的规则，**不要在这卷里重新发明数值含义**。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | string | 唯一 ID，例如 `ghost-shan-shou` |
| `name_cn` / `name_en` | string | 显示名（cn 是给玩家看的，例 "山顶守者"）|
| `所属段位` | int 1-6 | 对应登山路径哪一段 |
| `教学概念` | enum | 半按对焦 / 长曝消形 / 快门冻结 / 综合（Boss A）/ 长曝积累 / 综合（Boss B）|
| `外观` | 文本 | 显形时长得什么样（**人形 + 身份感 + 时间线索**，不是面具）|
| `行为 AI` | 枚举 + 文本 | 静坐 / 慢飘 / 掠过 / 风暴中静立 / 星空回望 / 山顶背对 |
| `移动速度` | m/s | 0 表示静止 |
| `blur_multiplier` | float | `CAMMotionBlurSubject.blurMultiplier`，默认 1 |
| `最低对焦距离` | m | 通常 3m，Boss B 段 6 可能要更近 |
| `亮度偏好` | 枚举 | 极暗 / 暗 / 中 / 亮 |
| `passingScore` | int 0-100 | 拍达标门槛 |
| `朋友笔记本对应页` | int 1-6 | 这只影所在段的笔记本第几页 |

## 治愈基调下影的设计指引（v2 必读）

1. **影是"被山记下来的人"**，不是怪物。每只影显形时玩家应该感到"这个人当年走到了这里"，而不是"我要躲它 / 打它"。
2. **不主动靠近玩家**。影通常静止 / 在自己的循环路径里 / 背对玩家。
3. **通常不发出威胁声**。可以发出环境声（脚步、风声、轻声哼唱），不是吓人的低吟。**例外**：诡异调味层允许偶发的"被跟踪"音效（远处人语 / 身后脚步），详见 [`45-uncanny.md`](45-uncanny.md)；事件级 jumpscare 全片仅段 4 风口 Boss A 首次命中 1 处。
4. **拍照不是"猎杀"**。拍达标 = "替山把这个人的样子留下来一份"。视觉反馈是温柔的（一道光从影身上飘起 / 影向玩家方向轻轻点头 / 慢慢淡出），不是凝固崩解。
5. **影对应朋友笔记本的某一页**。设计影时同时设计笔记本那页的内容（朋友当年遇到这只影时怎么拍 / 拍成什么样 / 留下什么话）。
6. **每只影都有一段"故事一句话"**：这个人是谁 / 什么时候走过 / 在这做了什么。**不必明说**，靠环境暗示（衣着 / 道具 / 姿态 / 笔记本里朋友写的旁白）。
7. **可选：诡异调味钩子**。影卡可以加一个可选段"§ 诡异调味钩子"，描述这只影在 [`45-uncanny.md`](45-uncanny.md) 7 类诡异里的角色（如"段 1 林祠静坐者承担观察反转"）。具体事件清单在段位卷 [`60-levels/01-shan.md`](60-levels/01-shan.md) 里集中描述，影卡只锚定。

## 与朋友笔记本的关系

每只影必须配套一页笔记本。建议格式：

```
笔记本 第 N 页
─────────────────
朋友的字迹（一段）：
"上次我在这里看见一个 ___ 。
 我用 ___ 快门 + ___ 光圈拍的。
 没拍清。如果你能拍下来 ___ 。"

朋友的样片（缺角 / 模糊）：
[一张 PNG，cel-shaded 风]

教的相机概念：
[这一段引入的参数旋钮 + 用法提示]
```

拍达标后，笔记本这页解锁"完整版"——朋友补完那段话 + 提供更清晰的样片 + 一段日记。

## Boss 影卡模板

复制整段到 `41-ghosts/ghost-xxx.md`：

````markdown
# ghost-xxx · 显示名（中文）

🟡 In-Progress

**分类**：Boss 影
**所属段位**：段 N（风口 / 顶峰）
**首次出现关卡**：[01 · 山](../60-levels/01-shan.md)

## 设定

- 外观：（**人形 + 身份感 + 时间线索**，不要面具 / 不要兽相）
- 故事一句话：这个人是谁 / 什么时候走过 / 在这做了什么
- 朋友与它的关联：朋友当年是否拍过 / 拍成什么样 / 留下什么话

## 行为

- AI 模式：（静立 / 风暴中不动 / 山顶背对 等）
- 显形条件：举相机 + 半按对焦命中 + 在该段位地理范围内
- 拍照后反应：温柔淡出 / 一道光飘起 / 向玩家点头

## 机械数值

| 项 | 值 | 备注 |
|---|---|---|
| `blur_multiplier` | 1.0 | |
| 推荐快门 | 1/X s | |
| 推荐光圈 | f/X | |
| 推荐 ISO | XXX | |
| 最低对焦距离 | 3 m | |
| 亮度偏好 | 暗 / 中 / 亮 | |
| 占屏比要求 | ≥ 12% | |
| **passingScore** | **65 / 70** | Boss A=65，Boss B=70 |

## 朋友笔记本对应页

第 N 页内容（拍糊版 + 拍达标版分别写）

## 拍达标的奖励

- 笔记本完整版解锁
- （Boss A）隐藏页 "我也没拍成，谢谢"
- （Boss B）完整结局
````
