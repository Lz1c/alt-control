# 45 · 诡异调味规则书（uncanny layer）

> 这一卷：v2 治愈基调下允许的**少量恐怖元素**和 SCP 风诡异感的**设计规则与边界**。
> 具体每段山路的事件清单见 [`60-levels/01-shan.md`](60-levels/01-shan.md)。
>
> **2026-05-01 v2 调味剂决议** — 详见 [`plan/decisions-log.md`](plan/decisions-log.md)。

## 为什么加这一卷

v2 收紧时反设计写明"不做 jumpscare / 不做恐怖"。事后用户回看觉得**纯治愈太平**，登山路上需要少量"山的代价 / 不对劲"作为情绪反差，否则 30-45 分钟会乏味。

但 v2 主调仍然是治愈——这一卷是**调味剂级别的护栏文件**：定死多少、什么形态、放在哪里、绝对不放在哪里。

## 三个边界（硬约束）

1. **山顶 Boss B 段绝对干净**——情感终点不能毁。所有诡异事件在段 6 顶峰之前结束。
2. **全片只允许 1 处真扑面 jumpscare**（音效 + 脸闪现），位置固定在段 4 风口 Boss A 的首次半按对焦命中。其他都是**积累式不安**，不是廉价惊吓。
3. **概率触发 + 不影响通关**——治愈玩家可以**完全错过**所有诡异事件仍然玩通；深度玩家会发现"这游戏不对劲"。双轨设计。

## 7 类诡异（SCP 风分类）

每只影卡 / 每段山路在描述诡异调味时**用这套分类标签**，方便程序实装和审计。

| 类型 | 内核 | 程序实装思路 |
|---|---|---|
| **被跟踪** | 你身后有东西，转身就停 | 3D 音源跟随玩家位置 + 看不见的 collider |
| **错位（物体挪位）** | 物体在你不看的时候动了 | GameObject 状态机：玩家视野外切换位置 |
| **观察反转** | 不举相机能瞄到，举相机就没了——和核心机制反一下 | Layer 切换 + 取景器渲染时禁用该 layer |
| **计数错乱** | 相册里多一张你不记得拍过的照片 | 拍照存盘时按概率加一张预制 PNG，时间戳穿插 |
| **时间感** | 远处段位的声音 / 自己刚走过的路上有自己的剪影 | 触发器跨段位音源 + 一次性 ghost player 模型 |
| **声音错位** | 风声里混入听不清的人语 / 不属于该段位的环境音 | AudioMixer 偶发层 |
| **照片诡异** | 拍下来的 PNG 里多一只手 / 多一个轮廓 | 拍照后处理 shader 注入随机叠加层 |

## 全片预算（硬上限）

| 项 | 上限 | 实际 |
|---|---|---|
| 真扑面 jumpscare | 1 | 1（风口 Boss A 首次命中） |
| 段位级别诡异事件 | 8 | 7（trailhead 1 / 段 1 1 / 段 2 3 / 段 3 1 / 段 4 1 / 段 5 2，段 6 = 0） |
| 全片散布的环境层 | 不限，但要克制 | 风声混入低语 / 雪地脚印短现 / 偶发咳嗽 |

## 设计原则

1. **不告诉玩家有诡异层**。Steam 标签写"治愈 / 摄影 / 步行模拟"，不写"恐怖"。诡异是隐藏味道，不是宣传卖点。
2. **诡异和叙事要咬合**。不是为了吓人而吓人——朋友死在这座山上，山有过太多过往登山者，"山记得太多"是合理的不安。
3. **每个诡异事件都该让玩家 doubt 自己**："我看错了吗？" 比"啊吓我一跳"高级。
4. **音效低于视觉**。低频 / 风的变形 / 远处的人语 > 突发刺响。**只有那 1 处扑面允许刺响**。
5. **不解释**。玩家不会通过笔记本 / 文案得到任何"这是因为 X"的解释。山就是山。
6. **可被理性玩家完全错过**。一周目正常玩通的玩家可能 0-2 次遇到诡异事件；二周目认真观察的玩家可能 5-7 次都遇到。

## 程序基础设施清单（W2-W3 实装）

W2 单点拉通段 1 林祠静坐者时，**优先做"观察反转"那条**作为本卷的垂直切片：

- [ ] `UncannyEvent`（基类）：携带触发条件 / 概率 / 一次性 vs 重复 / 视觉 + 音效 hooks
- [ ] `UncannyTrigger`（玩家视野检测器）：判定玩家是否看着 / 是否举相机 / 是否在该段位范围内
- [ ] `ObservationReverseObject`（观察反转专用）：举相机时禁用 / 不举时启用，Layer 切换实现
- [ ] `JumpscareTrigger`（仅风口）：一次性触发器，监听 Boss A 首次半按对焦命中
- [ ] `AmbientStalker`（环境层散布）：随机时间间隔生成"风声混入人语 / 远处咳嗽 / 雪地脚印短现"
- [ ] `FakePhotoInjector`（照片诡异）：拍照存盘时按概率注入预制 PNG（美术资产 W4 准备）
- [ ] `CrossSegmentAudio`（时间感）：跨段位音源叠加层

## 调性参考

- **SCP Foundation** 的网页文档调性：克制描述 + 留白 + 玩家自己脑补
- **风之旅人 Journey** 的暗黑长袍守卫者：威胁感来自"它在那里"而非"它扑过来"
- **Firewatch** 森林深处的不安瞬间：远处一个不该有的轮廓
- **Ghost of Tsushima · 冥人异闻**：治愈+诡异的并置先例
